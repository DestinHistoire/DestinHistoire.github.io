A.配对(排序+贪心)

https://ac.nowcoder.com/acm/contest/3007/A

题目描述

  现在有正整数集合 $A$ 和 $B$，每个集合里有 $n$ 个数，你要建立他们间的一一映射。

  将每对配对的数字相加可以得到 $n$ 个和，你要做的就是最大化第 $k$ 大的和 $(1\leq k\leq n≤100000)$。输入的所有数字不超过 $10^8$。

分析

  要使得第 $k$ 大的和尽可能大，显然可以贪心。组成这 $k$ 对数字的显然是 $A$ 中最大的 $k$ 个数字和 $B$ 中最大的 $k$ 个数字。

  问题转化为怎样配对使最小的和最大：如果 $A_1<A_2,B_1<B_2$ ，那么一定是由 $A_1$ 和 $B_2$ 配对较优。

  经过归纳可以得到，倒序配对是最优的。

代码

#include<bits/stdc++.h>
using namespace std;
int a[200010],b[200010],c[200010];
int main()
{
    int n,k;
    cin>>n>>k;
    for(int i=1;i<=n;i++)
        scanf("%d",&a[i]);
    for(int i=1;i<=n;i++)
        scanf("%d",&b[i]);
    sort(a+1,a+1+n);
    sort(b+1,b+1+n);
    for(int i=1;i<=k;i++)
        c[i]=a[n-k+i]+b[n-i+1];
    sort(c+1,c+1+k);
    cout<<c[1]<<endl;
    return 0;
}

C.汉诺塔(Dilworth定理+dp优化)

https://ac.nowcoder.com/acm/contest/3007/C

题目描述

  现在你有 $n$ 块矩形木板，第 $i$ 块木板的尺寸是 $x_i*y_i$，第 $i$ 块木板能放在第 $j$ 块木板上方当且仅当 $x_i<x_j$ 且 $y_i<y_j$，于是你很可能没法把所有的木板按照一定的次序叠放起来 $(1\leq n\leq 100000,1\leq x_i,y_i\leq n)$ 。

  你想把这些木板分为尽可能少的组，使得每组内的木板都能按照一定的次序叠放。给出任意一种合理的分组方案。

分析

  将木板按照 $x_i$ 从小到大排序，将这时的 $y_i$ 数列记为 $z_i$ 数列，则问题变成将 $z_i$ 划分为尽可能少的若干组上升子序列。

  根据 $Dilworth$ 定理，最小组数等于 $z_i$ 的最长下降子序列长度。

  要求最长下降子序列的长度，我们有一种经典的二分优化dp的方法。 借助这种做法能给出一种构造方法，在求出最小组数的同时得出方案。

  将状态数组的每个位置变为栈，用入栈操作代替修改元素操作，即可在求出组数的同时，用这些栈来完成对数列的划分。

代码

#include<bits/stdc++.h>
using namespace std;
struct node
{
    int x,y;
    int ID;
    bool operator < (const node &rhs)const{
        return x<rhs.x;
    }
}a[100010];
int ans[100010],sta[100010];
int main()
{
    int n,cnt=0;
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        scanf("%d%d",&a[i].x,&a[i].y);
        a[i].ID=i;
    }
    sort(a+1,a+1+n);
    for(int i=1;i<=n;i++)
    {
        int l=1,r=cnt;
        while(l<=r)
        {
            int mid=(l+r)/2;
            if(sta[mid]<a[i].y)
                r=mid-1;
            else
                l=mid+1;
        }
        sta[l]=a[i].y;
        if(l>cnt)
            cnt=l;
        ans[a[i].ID]=l;
    }
    cout<<cnt<<endl;
    for(int i=1;i<=n;i++)
        printf("%d ",ans[i]);
    cout<<endl;
    return 0;
}

D.重排列(思维)

https://ac.nowcoder.com/acm/contest/3007/D

题目描述

  一个序列的重排列是指对这个序列中的元素进行若干次（包括 $0$ 次）交换操作后得到的新序列。

  在本题中，序列中可能出现重复的数字，他们被视作不同的元素。例如，序列 $1\ 1$ 的重排列有两种。

  现在有两个长度为 $n$ 的非负整数序列 $A$ 和 $B$，问有多少种 $A$ 的重排列满足对于所有的 $1\leq i\leq n$，有 $a_i\leq b_i(1\leq n\leq 100000,0\leq A_i\leq B_i\leq 10^9)$。

  由于答案可能很大，只需要输出答案对 $10^9+7$ 取模的结果。

分析

  按升序将 $A$ 和 $B$ 排序不影响结果。按下标从小到大考虑 $A$ 的每个位置填什么数，而并不关心具体填了什么，所以只需要记录每一步有多少个数可填就可以了。

代码

#include<bits/stdc++.h>
using namespace std;
int a[200010],b[200010];
const int mod=1e9+7;
int main()
{
    int n;
    cin>>n;
    for(int i=1;i<=n;i++)
        scanf("%d",&a[i]);
    for(int i=1;i<=n;i++)
        scanf("%d",&b[i]);
    sort(a+1,a+1+n);
    sort(b+1,b+1+n);
    long long ans=1;
    for(int i=1;i<=n;i++)
    {
        int temp=upper_bound(a+1,a+1+n,b[i])-(a+1);
        ans=ans*(temp-i+1)%mod;
    }
    cout<<ans<<endl;
    return 0;
}

F.十字阵列(模拟)

https://ac.nowcoder.com/acm/contest/3007/F

题目描述

  小 Q 新学会了一种魔法，可以对一个 $n$ 行 $m$ 列的网格上的敌人造成伤害。

  第 $i$ 次使用魔法可以对网格上的一个十字形区域（即第 $x_i$ 行和第 $y_i$ 列的并）中的每个格子上的敌人造成 $z_i$ 点伤害

  现在小 Q 一共使用了 $h$ 次魔法，你需要在所有的施法完成之后统计造成伤害的情况，输出 $\displaystyle{\sum\limits_{i=1}^{n}\sum\limits_{j=1}^{m}}w_{ij}*(i+j)$。 $(1\leq h\leq 500000,1\leq x_i,y_i,z_i,n,m\leq 2000)$

分析

  $sumx[x_i]$ 统计对第 $x$ 行造成的伤害，$sumy[y_i]$ 统计对第 $y_i$ 行造成的伤害，$w[i][j]$ 统计对第 $i$ 行第 $j$ 列的格子造成的伤害，进行操作 $(x_i,y_i,z_i)$ 时，$sumx[x_i]+=z_i,sumy[y_i]+=z_i,w[x_i][y_i]-=z_i$，统计答案时，查询 $(sumx[i]+sumy[i]+w[i][j])*(i+j)$ 即可，

代码

#include<bits/stdc++.h>
using namespace std;
const int mod=1e9+7;
long long w[2010][2010];
long long sumx[2010],sumy[2010];
int main()
{
    int n,m,q;
    cin>>n>>m>>q;
    while(q--)
    {
        int x,y,val;
        scanf("%d %d %d",&x,&y,&val);
        sumx[x]=(sumx[x]+val)%mod;
        sumy[y]=(sumy[y]+val)%mod;
        w[x][y]=(w[x][y]-val+mod)%mod;
    }
    long long ans=0;
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
        {
            long long res=(sumx[i]+sumy[j]+w[i][j])%mod;
            ans=(ans+res*(i+j)%mod)%mod;
        }
    }
    cout<<ans<<endl;
    return 0;
}

G.括号序列(栈)

https://ac.nowcoder.com/acm/contest/3007/G

题目描述

合法括号序列的定义是：

$1.$空序列是合法括号序列；

$2.$如果 $S$ 是一个合法括号序列,那么 $(S)$ 是合法括号序列；

$3.$如果 $A$ 和 $B$ 都是合法括号序列，那么 $AB$ 是一个合法括号序列。

  现在给定一个括号序列，求最少删去几个括号能得到一个合法的括号序列 设括号序列的长度为 $n(1\leq n\leq 1000000)$（由于空串是合法的括号序列，所以答案可以是 $n$）。

分析

  括号序列合法的一个充要条件是： 设 $($ 为 $1$，$)$ 为 $-1$，则：①序列所有位置前缀和非负；② $($ 与 $)$ 数量相等。

  为了保证条件①，我们可以用栈模拟括号序列的匹配过程： 碰到一个 $($ 就加入栈中，碰到一个 $)$ 就消去栈顶的一个 $($， 如果栈中没有 $($ 则这个 $)$ 必须要删去。

  在这个过程结束之后，如果 $($ 比 $)$ 多，则从后向前删去多余的 $($，直到序列合法即可。

代码

#include<bits/stdc++.h>
using namespace std;
char str[1000010];
int main()
{
    int T;
    cin>>T;
    while(T--)
    {
        int n;
        cin>>n;
        scanf("%s",str+1);
        int l=0,r=0,ans=0,sum=0;
        for(int i=1;i<=n;i++)
        {
            if(str[i]=='(')
                l++,sum++;
            else if(sum>0)
                r++,sum--;
            else    
                ans++;
        }
        cout<<ans+l-r<<endl;
    }
    return 0;
}

J.签到题(解方程)

https://ac.nowcoder.com/acm/contest/3007/J

题目描述

  现有一个边长为正整数的三角形，问能否以其三个顶点为圆心画三个圆，使三个圆两两外切 ，三边长均不超过 $10^8$。

  如果三条边不能构成三角形，输出 $wtnl$。

  如果三条边能构成三角形但不能画出符合要求的圆，输出 $No$。

  否则输出一行 $Yes$。然后在第二行输出一组方案，按升序给出三个圆的半径，保留两位小数。

分析

  假设三角形边长分别为 $a,b,c$，三个圆的半径分别为 $r_1,r_2,r_3$。

  $r_1=\frac{a-b+c}{2},r_2=\frac{a+b-c}{2},r_3=\frac{-a+b+c}{2}$

  由于三角形三边之和大于第三边，因此 $r_1,r_2,r_3>0$，不存在 $No$ 的情况。

代码

#include<bits/stdc++.h>
using namespace std;
double a[10],r[10];
int main()
{
    cin>>a[1]>>a[2]>>a[3];
    sort(a+1,a+4);
    if(a[3]>=a[1]+a[2])
        puts("wtnl");
    else
    {
        puts("Yes");
        r[1]=(a[1]-a[2]+a[3])/2.0;r[2]=(a[1]+a[2]-a[3])/2.0;r[3]=(-a[1]+a[2]+a[3])/2.0;
        sort(r+1,r+4);
        printf("%.2lf %.2lf %.2lf",r[1],r[2],r[3]);
    }
    return 0;
}
