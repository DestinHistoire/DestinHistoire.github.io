乘法逆元
数论 逆元
算法笔记
发布日期:   2020-02-26

费马小定理

如果 $p$ 是一个质数，且 $0<a<p$ ，则 $a^{p-1}\equiv 1\pmod p$。

证明：

  $1\ast a\ \text{mod}\ p=a_1$

  $2\ast a\ \text{mod}\ p=a_2$

  $\cdots\cdots$

  $(p-1)\ast a\ \text{mod}\ p=a_{p-1}$

易知，$i\neq j$ 时，$a_i\neq a_j$，则 $a_1,a_2,\cdots,a_{p-1}$ 为 $1,2,3,\cdots,p-1$ 的一个全排列。

将上式两边全部乘在一起，得到：$(p-1)!\ast a^{p-1}\ \text{mod}\ p=(p-1)!$

即 $a^{p-1}\equiv 1\pmod p$，得证。

逆元的定义

  若整数 $ b,p $ 互质，并且 $ b\mid a $ ，则存在一个整数 $x$，使得 $ \frac{a}{b}\equiv a\ast x(\text{mod}\ p) $。称 $x$ 为 $ b $ 的模 $ p $ 乘法逆元，记为 $ b^{-1}(\text{mod}\ p)$。

  $\because \frac{a}{b}\equiv a\ast b^{-1}\equiv \frac{a}{b}\ast b\ast b^{-1}(\text{mod}\ p)$。

  $\therefore b\ast b^{-1}\equiv1(\text{mod}\ p)$。

  如果 $ p $ 是质数，并且 $ b<p $，根据费马小定理，$ b^{p-1}\equiv 1(\text{mod}\ p) $，即 $ b\ast b^{p-2}\equiv1(\text{mod}\ p) $。

  因此，当模数 $p$ 为质数时，$ b^{p-2} $ 即为 $b$ 的乘法逆元 。

  如果只是保证 $b,p$ 互质，那么乘法逆元可通过 *求解同余方程 $ b\ast x\equiv1(\text{mod}\ m)$ * 得到。

  有了乘法逆元，我们在计数类问题中即使遇到 $\frac{a}{b}$ 这样的乘法算式，也可以先把 $a,b$ 各自对模数 $p$ 取模，再计算 $a* b^{-1}\ \text{mod}\ p$ 作为最终的结果。当然，前提是必须保证 $b,p$ 互质（当 $p$ 是质数时，等价于 $b$ 不是 $p$ 的倍数）

求逆元的三种方法

方法一 扩展欧几里得算法

  当模数 $ p $ 比较大时，这种方法比后面介绍的几种方法快。

  这种方法就是利用扩展欧几里得算法求解线性同余方程 $a\ast x\equiv c(\text{mod}\ p)$ 的 $c=1$ 的情况。我们可以转化为 $a* x+p* y=1$ ，求这个方程的解。

  这种方法还有个好处在于，当 $ a,p $ 互质，但 $ p $ 不是质数的时候也可以使用。

#include<bits/stdc++.h>
using namespace std;
int x,y;
void exgcd(int a,int b)
{
    if(b==0)
    {
        x=1;
        y=0;
        return ;
    }
    exgcd(b,a%b);
    int temp=x;x=y;y=temp-a/b*y;
}
int main()
{
    int n,p;
    cin>>n>>p;
    for(int i=1;i<=n;++i)
    {
        exgcd(i,p);
        cout<<(x%p+p)%p<<endl;
    }
    return 0;
}

方法二 费马小定理

  费马小定理：若 $p$ 为素数，$a$ 为正整数，且 $a$、$p$ 互质。 则有 $a^{p-1} \equiv 1 (\text{mod}\ p)$。

  可以发现它这个式子右边刚好为 $1$。

  所以我们可以放入原式，得到：
$$
a\ast x \equiv 1 \pmod p\Longrightarrow a\ast x\equiv a^{p-1} \pmod p\Longrightarrow x \equiv a^{p-2} \pmod p
$$
  所以可以用快速幂来算出 $ a^{p-2} \pmod p$ 的值，这个数就是 $a$ 的逆元。

#include<bits/stdc++.h>
using namespace std;
long long quick_pow(long long a,long long b)
{
    long long ans=1;
    a%=p;
    while(b)
    {
        if(b&1) 
            ans=ans*a%p;
        a=a*a%p;
        b>>=1;
    }
    return ans%p;
}
int main()
{
    int n,p;
    cin>>n>>p;
    for(int i=1;i<=n;i++)
        cout<<quick_pow(i,p-2)<<endl;
    return 0;
}

方法三 线性递推

  首先，$ 1^{-1}\equiv1\pmod p $

  然后，设 $ p=k\ast i + r (r<i,1<i<p) $，再将这个式子放到模 $ p $ 意义下就会得到：

  $k* i+r\equiv0\pmod p $

  在两边同时乘上 $i^{-1},r^{-1}$ 就会得到：

  $ k\ast r^{-1}+i^{-1}\equiv 0\pmod p$

  $i^{-1}≡-k* r^{-1}\pmod p$

  $i^{-1}\equiv -\lfloor\frac{p}{i}\rfloor * (p\ \text{mod}\ i)^{-1}\pmod p$

  于是，就可以从前面推出当前的逆元了。代码只有两行： 

inv[i]=-(p/i)*inv[p%i];
inv[i]=(inv[i]%p+p)%p;

  实际上，这也提供了一种 $ O(\log_2 p) $ 的时间内求出单个数逆元的方法，只要直接按照上述公式递归就可以了。可以证明：$ p\ \text{mod}\ i<i/2 $，每次递归问题规模减半，最终只会有 $ O(\log_2 p)$ 次递归。

#include<bits/stdc++.h>
using namespace std;
long long n,p,inv[3000010];
void init(int n,int p)//线性求逆元，时间复杂度O(n) 
{
    inv[1]=1;
    for(int i=2;i<=n;i++)
    {
        inv[i]=-(p/i)*inv[p%i];
        inv[i]=(inv[i]%p+p)%p;
    }
}
int main()
{
    cin>>n>>p;
    init(n,p); 
    for(int i=1;i<=n;i++)
        printf("%lld\n",inv[i]);
    return 0;
}
