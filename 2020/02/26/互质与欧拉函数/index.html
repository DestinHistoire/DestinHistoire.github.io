互质与欧拉函数
数论 欧拉函数
算法笔记
发布日期:   2020-02-26

互质的定义

  $\forall a,b\in N $，若 $\gcd(a,b)=1 $，则称 $a,b$ 互质。

  对于三个数或更多个数的情况，我们把 $ \gcd(a,b,c)=1$ 的情况称为 $a,b,c$ 互质。把 $\gcd(a,b)=\gcd(a,c)=\gcd(b,c)=1 $ 称为 $ a,b,c $ 两两互质。后者显然是一个更强的条件。

欧拉函数

  $1$ ~ $N$ 中与 $N$ 互质的数的个数被称为欧拉函数，记为 $\varphi(N)$。

性质1

  若 $p$ 为质数， 则：$\varphi(p)=p-1$。

证明

  质数 $p$ 除了 $1$ 以外的因数只有 $p$，故 $1$ 至 $p$ 的整数只有 $p$ 与 $p$ 不互质。

性质2

  若 $p$ 为质数，对于某一个素数 $p$ 的幂次 $ p^k $，有：$\varphi(p^k)=(p-1)* p^{k-1}$

证明

  $\because$ 比 $ p^{k} $ 小的正整数有 $ p^k-1 $ 个。其中，所有能被 $p$ 整除的数字可以表示成 $p* t=(t=1,2,···,p^{k-1}-1)$，即共有 $p^{k-1}-1$ 个这样的数字能被 $p$ 整除，从而不与 $ p^k $ 互质。
  $\therefore$ $\varphi(p^a)=p^a-1-(p^{a-1}-1)=(p-1)* p^{a-1}$

性质3

  若 $ \gcd(a,b)=1 $，则 $ \varphi(a\ast b)=\varphi(a)\ast\varphi(b) $

证明

  在比 $a* b$ 小的 $a\ast b-1$ 个整数中，只有那些既与 $a$ 互质（有$\varphi(a)$个），又与 $b$ 互质（有$\varphi(b)$个）的数字才会与 $a* b$ 互质。显然满足这种条件的数字有 $\varphi(a)\ast \varphi (b)$ 个。把该性质推广到一般的函数上，可以得到“积性函数”的概念。

性质4

  若在算术基本定理中，$ N=p_1^{c_1}p_2^{c_2}\cdots p_m^{c_m} $，则：$\varphi(N)=N\ast\frac{p_1-1}{p_1}\ast \frac{p_2-1}{p_2}\ast\cdots\ast\frac{p_m-1}{p_m}=N* \prod\limits_{p|N}(1-\frac{1}{p})$

证明一

  由于各个质数幂互相之间是互质的，根据性质 $1,2,3$ 得出：

证明二

  设 $p$ 是 $N$ 的质因子，$1$ ~ $N$ 中 $p$ 的倍数有 $ p,2p,3p,\cdots,(N/p)\ast p $，共 $ N/p $ 个。

  同理，若 $q$ 也是 $N$ 的质因子，则 $1$ ~ $N$ 中 $ q $ 的倍数有 $ N/q $个。如果我们把这 $ N/p+N/q $ 个数去掉，那么 $ p\ast q $ 的倍数被排除了两次,需要加回来一次。因此，$1$ ~ $N$ 中不与 $ N $ 含有共同质因子 $p$ 或 $q$ 的数的个数为：$N-\frac{N}{p}-\frac{N}{q}+\frac{N}{pq}=N* (1-\frac{1}{p}-\frac{1}{q}+\frac{1}{pq})=N(1-\frac{1}{p})(1-\frac{1}{q})$ 。

  上述思想被称为容斥原理。类似地，可以在 $N$ 的全部质因子上使用容斥原理，即可得到 $1$ ~ $N$ 中不与 $N$ 含有任何共同质因子的数的个数，也就是与 $N$ 互质的数的个数。

  证毕。

  根据欧拉函数的计算式，我们只需要分解质因数，即可顺便求出欧拉函数。

int phi(int n)
{
    int ans=n;
    for(int i=2;i*i<=n;i++)
    {
        if(n%i==0)
        {
            ans=ans/i*(i-1);
            while(n%i==0)
                n=n/i;
        }
    }
    if(n>1)
        ans=ans/n*(n-1);
    return ans;
}

性质5

  $\forall n>1$，$1{\sim}n$ 中与 $n$ 互质的数的和为 $\frac{n\ast \varphi(n)}{2}$。

证明

  $\because \gcd(n,x)=\gcd(n,n-x)$

  $\therefore$ 与 $ n $ 不互质的数 $ x,n-x $ 成对出现，平均值为 $ \frac{n}{2} $
  $\therefore$ 与 $n$ 互质的数的平均值也是 $ \frac{n}{2} $，进而得到性质 $5$。

积性函数

性质

  若 $f$ 是积性函数，且在算术基本定理中 $n=\prod\limits_{i=1}^mp_i^{c_i}$，则 $f(n)=\prod\limits_{i=1}^mf(p_i^{c_i})$

证明

  把 $n$ 分解质因数，按照积性函数的定义，该性质显然成立。

性质6

  设 $p$ 为质数，若 $p|n$ 且 $p^2|n$，则 $\varphi(n)=\varphi(\frac{n}{p})* p$。

证明

  若 $p|n$ 且 $p^2|n$，则 $n,\frac{n}{p}$ 包含相同的质因子，只是 $p$ 的指数不同。直接把 $\varphi(n)$ 与 $\varphi(\frac{n}{p})$ 按照欧拉函数的计算公式写出，两者相除，商为 $p$，所以性质 $6$ 成立。

性质7

  设 $p$ 为质数，若 $p|n$ 但 $p^2\nmid n$，则 $\varphi(n)=\varphi(\frac{n}{p})* (p-1)$。

证明

  若 $p|n$ 但 $p^2\nmid n$，则 $\gcd(p,\frac{n}{p})=1$，由 $\varphi$ 是积性函数得 $\varphi(n)=\varphi(\frac{n}{p})* \varphi(p)$ ，而 $\varphi(p)=p-1$，所以性质 $7$ 成立。

性质8

  $\sum\limits_{d|n}\varphi(d)=n$

证明

  设 $f(n)=\sum\limits_{d|n}\varphi(d)$。用乘法分配律展开比较，再利用 $\varphi$ 是积性函数，得到：若 $n,m$ 互质，则 $f(nm)=\sum\limits_{d|nm}\varphi(d)=(\sum\limits_{d|n}\varphi(d))* (\sum\limits_{d|m}\varphi(d))=f(n)* f(m)$。即 $\sum\limits_{d|n}\varphi(d)$ 是积性函数。对于单个质因子，$f(p^m)=\sum\limits_{d|p^m}\varphi(d)=\varphi(1)+\varphi(p)+\varphi(p^2)+···+\varphi(p^m)$ 是一个等比数列求和再加 $1$，结果为 $p^m$。所以 $f(n)=\prod\limits_{i=1}^mf(p_i^{c_i})=\prod\limits_{i=1}^m p_i^{c_i}=n$，性质 $8$ 成立。

欧拉函数的线性筛法

  该算法可以在线性时间内筛素数的同时求出所有数的欧拉函数。

  由前面提到的性质 $6$ 和性质 $ 7 $：

  $1.$设 $p$ 为质数，若 $p|n$ 且 $p^2|n$，则 $\varphi(n)=\varphi(\frac{n}{p})\ast p $。

  $2.$设 $p$ 为质数，若 $p|n$ 但 $p^2\nmid n$，则 $\varphi(n)=\varphi(\frac{n}{p})\ast(p-1) $。

  在线性筛法中，每个合数 $n$ 只会被它的最小质因子 $p$ 筛一次。恰好可以在此时执行上面两条判断，从 $\varphi(\frac{n}{p})$ 递推到 $\varphi(n)$。

int vis[N],prime[N],phi[N];
void euler(int n)
{
    memset(vis,0,sizeof(vis));//最小质因子
    cnt=0;//质数数量
    for(int i=2;i<=n;i++)//相当于分解质因式的逆过程 
    { 
        if(vis[i]==0)//i是质数 
        {
            vis[i]=i;
            prime[++cnt]=i;
            phi[i]=i-1;//当i是质数时,phi[i]=i-1
        }
        for(int j=1;j<=cnt;j++)//给当前的数i乘上一个质因子
        {
            if(prime[j]>vis[i]||prime[j]>n/i)
                break;//i有比prime[j]更小的质因子,或者超出n的范围,停止循环
            vis[i*prime[j]]=prime[j];//prime[j]是合数i*prime[j]的最小质因子 
            if(i%prime[j]==0)//判断prime[j]是否是i的约数 
                phi[i*prime[j]]=phi[i]*prime[j];
            else
                phi[i*prime[j]]=phi[i]*(prime[j]-1);//这里prime[j]-1就是phi[prime[j]],利用了欧拉函数的积性 
        }
    } 
}
