A.做游戏(数学)

https://ac.nowcoder.com/acm/contest/3003/A

题目描述

  牛牛和牛可乐在玩石头剪刀布。牛牛和 牛可乐进行了多轮游戏，牛牛总共出了 $A$ 次石头，$B$ 次剪刀，$C$ 次布；牛可乐总共出了 $X$ 次石头，$Y$ 次剪刀，$Z$ 次布。 你需要求出牛牛最多获胜多少局。$(A+B+C=X+Y+Z, 0\leq A,B,C,X,Y,Z\leq 10^9)$

分析

尽可能让牛牛的每次出 剪刀/石头/布 对应到 牛可乐出 布/剪刀/石头。

$\min(A,Y)+\min(B,Z)+\min(C,X)$ 。时间复杂度 $O(1)$。

代码

#include<bits/stdc++.h>
using namespace std;
int main()
{
    int a,b,c;
    int x,y,z;
    cin>>a>>b>>c;
    cin>>x>>y>>z;
    long long ans=min(a,y);
    ans=ans+min(b,z);
    ans=ans+min(c,x);
    cout<<ans<<endl;
    return 0;
}

B.排数字(贪心)

https://ac.nowcoder.com/acm/contest/3003/B

题目描述

给定一个纯数字的字符串 $S$，在可以任意打乱 $S$ 顺序的情况下，求最多有多少个不同的子串为 $616$ 。

当两个子串 $S[l_1\dots r_1],S[l_2\dots r_2]$ 满足 $l_1\neq l_2$ 或 $r_1\neq r_2$ 时它们被认为是不同的。

分析

  $1$ 和 $666$ 以外不需要考虑。要让 $616$ 子串最多一定是 $61616\dots$ ，这样后面的串可以用前面的 $6$ ，数量为 $\min(\text{cnt}_6-1, \text{cnt}_1)$ （可以理解为前面一个 $6$ 后面 $16$ 循环）。时间复杂度 $O(∣S∣)$。

代码

#include <bits/stdc++.h>
using namespace std;
char str[200010];
int cnt[10],n;
int main()
{
    cin>>n;
    cin>>str;
    for(int i=0;i<n;i++)
        cnt[str[i]-'0']++;
    cout<<min(cnt[6]-1,cnt[1])<<endl;
    return 0;
}

C.算概率(数学+dp)

https://ac.nowcoder.com/acm/contest/3003/C

题目描述

  牛牛刚刚考完了期末，尽管 牛牛 做答了所有 $n$ 道题目，但他不知道有多少题是正确的。不过，牛牛知道第 $i$ 道题的正确率是 $p_i$​。
  牛牛想知道这 $n$ 题里恰好有 $0,1,\dots,n$ 题正确的概率分别是多少，对 $10^9+7$ 取模。
  对 $10^9+7$ 取模的含义是：对于一个 $b\neq 0$ 的不可约分数 $\frac{a}{b}$​，存在 $q$ 使得 $b\times q \bmod (10^9+7)$ ，$q$ 即为 $\frac{a}{b}$ 对 $10^9+7$ 取模的结果。

输入描述

  第一行，一个正整数 $n$ 。

  第二行，$n$ 个整数 $p_1,p_2,\dots,p_n$，在模 $10^9+7$ 意义下给出。保证 $1\leq n\leq 2000$。

输出描述

  输出一行 $n+1$ 个用空格隔开的整数表示答案（对 $10^9+7$ 取模）。

分析

$dp[i][j]$ 表示前 $i$ 道题做对 $j$ 道的概率。

转移时考虑第 $j$ 道题是否做对，转移方程为： $dp[i][j]=dp[i-1][j]\times (1-p_i)+dp[i-1][j-1]\times p_i $。
时间复杂度 $O(n^2)$ 。

代码

#include <bits/stdc++.h>
using namespace std;
const int mod=1e9+7;
long long p[2010];
long long dp[2010][2010];
int main()
{
    int n;
    cin>>n;
    for(int i=1;i<=n;i++)
        scanf("%lld",&p[i]);
    dp[0][0]=1;//前i个题答对j个
    for(int i=1;i<=n;i++)
    {
        dp[i][0]=dp[i-1][0]*(mod+1-p[i])%mod;//第i道题答错
        for(int j=1;j<=i;j++)
            dp[i][j]=(dp[i-1][j-1]*p[i]%mod+dp[i-1][j]*(mod+1-p[i])%mod)%mod;
    }
    for(int i=0;i<=n;i++)
        cout<<dp[n][i]%mod<<" ";
    cout<<endl;
    return 0; 
}

D.数三角(暴力)

https://ac.nowcoder.com/acm/contest/3003/D

题目描述

平面上有 n 个不重合的点，求 $n$ 个点形成的三角形中，总共有多少个钝角三角形。

保证 $1\leq n\leq 500$，$-10^4\leq x_i,y_i\leq 10^4$，任意两点不重合。

分析

  一个三角形的三边长 $a,b,c$ （ $c$ 最长 ）满足 $a^2+b^2<c^2$ （或存在两条边向量的点积 $<0$ ），则该三角形为钝角三角形。

  枚举三个点判断即可，注意判断共线和不要算重。

  时间复杂度 $O(n^3)$ 。

代码

#include<bits/stdc++.h>
using namespace std;
int x[510],y[510];
bool check(int i,int j,int k)
{
    return ((x[j]-x[i])*(x[k]-x[i])+(y[j]-y[i])*(y[k]-y[i])<0)&&((x[j]-x[i])*(y[k]-y[i])-(x[k]-x[i])*(y[j]-y[i])!=0);
}
int main()
{
    int n,ans=0;
    cin>>n;
    for(int i=1;i<=n;i++)
        scanf("%d %d",&x[i],&y[i]);
    for(int i=1;i<=n;i++)
        for(int j=i+1;j<=n;j++)
            for(int k=j+1;k<=n;k++)
                if(check(i,j,k)||check(k,i,j)||check(j,i,k))
                    ans++;
    cout<<ans<<endl;
    return 0;
}

E.做计数(数学)

https://ac.nowcoder.com/acm/contest/3003/E

题目描述

求有多少个不同的正整数三元组 $(i,j,k)$ 满足 $\sqrt i+\sqrt j=\sqrt k$，且 $i\times j\leq n$。$(1\leq n\leq 4\times10^7)$

当两个三元组 $(i_1,j_1,k_1),(i_2,j_2,k_2)$ 满足 $i_1\neq i_2$ 或 $j_1\neq j_2$ 或 $k_1\neq k_2$ 时它们被认为是不同的。

分析

  两边平方，得 $i+j+2\sqrt{ij}=k$，显然仅当 $i,j$ 都是整数且 $ij$ 为完全平方数时才会对应一个符合条件的 $k$ 。

  枚举 $[1,n]$ 中所有的完全平方数（完全平方数可以表示为 $x^2$ ，只需要枚举 $x$ ），再枚举这个完全平方数的因数，统计答案。

  $O(\sqrt n)$ 枚举完全平方数 $O(\sqrt n)$ 枚举因数，时间复杂度 $O(n)$。

代码

#include<bits/stdc++.h>
using namespace std;
int main() 
{
    int n,ans=0;
    cin>>n;
      for(int i=1;i*i<=n;i++)
      {
        for(int j=1;j<=i;j++)
              if(i*i%j==0)
                ans=ans+2;
        ans--;
    }
    cout<<ans<<endl;
      return 0;
}

F.拿物品(博弈+贪心)

https://ac.nowcoder.com/acm/contest/3003/F

题目描述

  牛牛和牛可乐面前有 $n$ 个物品，这些物品编号为 $1,2,\dots,n$ ，每个物品有两个属性 $a_i, b_i$。牛牛与牛可乐会轮流从剩下物品中任意拿走一个， 牛牛先选取。

  设牛牛选取的物品编号集合为 $H$，牛可乐选取的物品编号的集合为 $T$，取完之后，牛牛得分为$\sum_{i\in H} a_i$；而 牛可乐得分为 $\sum_{i\in T} b_i$。

  牛牛和 牛可乐都希望自己的得分尽量比对方大（即最大化自己与对方得分的差）。

  求出两人都使用最优策略的情况下，最终分别会选择哪些物品，若有多种答案或输出顺序，输出任意一种。

分析

  假设物品已经被选完，此时牛牛选择的物品 $A$ 属性的价值和是 $N$ ，牛可乐选择的物品 $B$ 属性价值和是 $M$。

  如果 牛牛的 $(a_1,b_1)$ 物品与牛可乐的 $(a_2,b_2)$ 交换，则 $N’=N-a_1+a_2, M’=M+b_1-b_2$，对于 牛牛（目标是最大化 $N-M$ ）来说会变得更优仅当 $a_1+b_1<a_2+b_2$（ $N’-M’>N-M$ 化简就能得到），对于牛可乐也一样。所以两人都会优先选择 $a_i+b_i$ 最大的物品。

  将物品按照两个属性的和从大到小排序，依次分给两人即可。

  除排序时间复杂度 $O(n)$ 。

代码

#include<bits/stdc++.h>
using namespace std;
struct node
{
    int a,b,ID;
    bool operator<(const node &rhs) const{
        return a+b>rhs.a+rhs.b;
    }
}num[200010];
int ans1[200010],ans2[200010];
int main()
{
    int n,cnt1=0,cnt2=0;
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        num[i].ID=i;
        scanf("%d",&num[i].a);
    }
    for(int i=1;i<=n;i++)
        scanf("%d",&num[i].b);
    sort(num+1,num+1+n);
    for(int i=1;i<=n;i++)
    {
        if(i%2==1)
            ans1[++cnt1]=num[i].ID;
        else
            ans2[++cnt2]=num[i].ID;
    }
    for(int i=1;i<=cnt1;i++)
        printf("%d ",ans1[i]);
    cout<<endl;
    for(int i=1;i<=cnt2;i++)
        printf("%d ",ans2[i]);
    cout<<endl;
    return 0;
}

H.施魔法(dp)

https://ac.nowcoder.com/acm/contest/3003/H

题目描述

  有 $n$ 个元素(编号 $1\dots n$ )，第 $i$ 个元素的能量值为 $a_i$。可以选择至少 $k$ 个元素来施放一次魔法，魔法消耗的魔力是这些元素能量值的极差。形式化地，若所用元素编号集合为 $S$，则消耗的魔力为 $\max_{i\in S}{a_i}-\min_{i\in S}{a_i}$。要求每个元素必须被使用恰好一次。求最少需要多少魔力才能用完所有元素。$(1\leq k\leq n\leq 3\times 10^5,0\leq a_i\leq 10^9)$

分析

  先将元素按能量值排序，可以证明存在一个最优方案，满足每个魔法一定消耗一段连续的元素。

  $dp[i]$ 表示用掉前 $i$ 个元素的最小代价。

  状态转移方程：$dp[i]=\min\limits_{j\in[1,i-k+1]} {dp[j-1]+a[i]-a[j]}=\min\limits_{j\in[1,i-k+1]} {dp[j-1]-a[j]}+a[i]$

  维护 $dp[i-1]-a[i]$ 的前缀最小值可以 $O(1)$ 转移。

  dp过程时间复杂度 $O(n)$。

代码

#include<bits/stdc++.h>
using namespace std;
int dp[300010],pre,a[300010];
int main()
{
    int n,k;
    cin>>n>>k;
    for(int i=1;i<=n;i++)
        scanf("%d",&a[i]);
    sort(a+1,a+n+1);
    pre=pre-a[1];
    for(int i=1;i<=k-1;i++)
        dp[i]=0x3f3f3f3f;
    for(int i=k;i<=n;i++)
    {
        dp[i]=pre+a[i];
        pre=min(pre,dp[i-k+1]-a[i-k+2]);
    }
    cout<<dp[n]<<endl;
    return 0;
}

I.建通道(位运算+贪心)

https://ac.nowcoder.com/acm/contest/3003/I

题目描述

  有 $n$ 个星球，第 $i$ 个星球有权值 $v_i$。第 i 个星球与第 j 个星球的之间建立传送通道的花费是 $\text{lowbit}(v_i\oplus v_j)$，其中 $\oplus$ 为二进制异或，而 $\text{lowbit}(x)$ 为 $x$ 二进制最低位 $1$ 对应的值，要使这 $n$ 个星球相互可达，需要的花费最少是多少。$(1\leq n\leq 2\times10^5,0\leq v_i < 2^{30})$。

分析

  首先将权值去重（权值相等的点连接代价为 $0$ ），设去重后有 $m$ 个点，接下来找到最小的二进制位 $k$ ，满足存在 $v_i$ 的这个二进制位是 $0$ 且存在 $v_j$ 的这个二进制位是 $1$ ，答案就是 $2^k\times (m-1)$ （相当于所有这位是 $0$ 的点与 $j$ 点连边，是 $1$ 的点与 $i$ 点连边）。

  排序和去重以外时间复杂度 $O(n+\log V)$。

代码

#include<bits/stdc++.h>
using namespace std;
int v[200010];
long long lowbit(int x){return x&-x;}
int main()
{
    int n,temp1=0xffffffff,temp2=0;
    cin>>n;
    for(int i=1;i<=n;i++)
        scanf("%d",&v[i]);
    sort(v+1,v+1+n);
    int m=unique(v+1,v+1+n)-(v+1);
    for(int i=1;i<=m;i++)
    {
        temp1=temp1&v[i];//找到m个数字中均为1的数位，说明存在数位为1
        temp2=temp2|v[i];//找到m个数字中异或后为0的数位，如果异或后为0，说明存在数位为0
    }
    cout<<(long long)(m-1)*lowbit(temp1^temp2)<<endl;
    return 0;
}

J.求函数(线段树区间合并)

https://ac.nowcoder.com/acm/contest/3003/J

题目描述

有 $n$ 个一次函数，第 $i$ 个函数为 $f_i(x)=k_i\times x+b_i$。

有 $m$ 次操作，每次操作为以下二者其一：

$\text{1 i k b}$ 将 $f_i(x)$ 修改为 $f_i(x)=k\times x+b$。

$\text{2 l r}$ 求 $f_r\left(f_{r-1}\left(\cdots\left(f_{l+1}\left(f_l(1)\right)\right)\cdots\right)\right)$。
答案对 $10^9+7$ 取模。

输入描述

第一行，两个正整数 $n,m$。
第二行，$n$ 个整数 $k_1,k_2,\dots,k_n$。

第三行，$n$ 个整数 $b_1,b_2,\dots,b_n$。

接下来 $m$ 行，每行一个操作，格式见上。$(1\leq n,m\leq 2\times 10^5,0\leq k_i,b_i<10^9+7)$

输出描述

对于每个求值操作，输出一行一个整数，表示答案。

分析

$f_r\left(f_{r-1}\left(\cdots\left(f_{l+1}\left(f_l(1)\right)\right)\cdots\right)\right)=\prod\limits_{i=l}^{r}k_i+\sum\limits_{i=l}^{r}b_i\prod\limits_{j=i+1}^{r}k_j$，当 $i+1>r$ 时，视 $\prod\limits_{j=i+1}^{r}k_j=1$
