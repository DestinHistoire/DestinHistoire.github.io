A.模板(贪心)

https://ac.nowcoder.com/acm/contest/3006/A

题目描述

  给定长度为 $n$ 的字符串 $s_1$ 和长度为 $m$ 的字符串 $s_2$。$(1\leq n,m\leq 10^5)$

  通过以下操作用尽可能少的步骤把一个密钥转换为另一个：

  $1.$ 将其中任意一个字母替换为另一个

  $2.$ 把最后一个字母删除

  $3.$ 在尾部添加一个字母

  求最少操作次数。

分析

  先考虑两个字符串相同的部分，显然，只要把原字符串和新字符串不同的字符修改就行了；对于两个字符串后面不同的部分，如果原字符串比较长，就全部删掉；如果原字符串比较短，就对照第二个字符串添加字符。

代码

#include<bits/stdc++.h>
using namespace std;
char s[100010],t[100010];
int main()
{
    int n,m;
    cin>>n>>m;
    scanf("%s",s+1);
    scanf("%s",t+1);
    int minn=min(n,m),ans=0;
    for(int i=1;i<=minn;i++)
        if(s[i]!=t[i])
            ans++;
    cout<<ans+max(n,m)-min(n,m)<<endl;
    return 0;
}

B.牛牛战队的比赛地(三分)

https://ac.nowcoder.com/acm/contest/3006/B

题目描述

  二维平面上有 $n$ 个点，坐标为 $(x_i,y_i)$ 。在 $x$ 轴上找一个点，使该点到其他点的最大距离最小。$(1\leq n\leq 100000,-10000\leq x,y\leq 10000)$

分析

  从 $x$ 轴的负无穷到 $x$ 轴的正无穷，最大距离显然是先减后增的函数，三分即可。

代码

#include<bits/stdc++.h>
using namespace std;
const int INF=0x3f3f3f3f;
const double eps=1e-7;
int n,x[100010],y[100010];
double dis(double temp, int i)
{
    return sqrt((temp-x[i])*(temp-x[i])+y[i]*y[i]);
}
double check(double temp)
{
    double ans=0;
    for(int i=1;i<=n;i++)
        ans=max(ans,dis(temp,i));
    return ans;
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
        scanf("%d %d",&x[i],&y[i]);
    double l=-1.0*INF,r=1.0*INF;
    while(r-l>eps)
    {
        double lmid=l+(r-l)/3.0;
        double rmid=r-(r-l)/3.0;
        if(check(lmid)>check(rmid))
            l=lmid;
        else
            r=rmid;
    }
    printf("%.7lf\n",check(r));
    return 0;
}

D.牛牛与牛妹的约会(贪心)

https://ac.nowcoder.com/acm/contest/3006/D

题目描述

  牛牛的坐标为 $x=a$，牛妹的坐标为 $x=b$。牛牛除了可以以 $1$ 单位距离/单位时间的速度移动任意时间以外，还可以花费 $1$ 单位时间进行闪现。每次闪现时，如果当前他的坐标是 $x=k$ ，他将闪现到$x=\sqrt[3]{k}$ 的位置。 求最短需要多少时间，牛牛可以找到牛妹。$(|a|,|b|\leq 10^6)$

分析

  比较直接走和闪现哪一个可以走的更快，选择更快的方法去走即可。

代码

#include<bits/stdc++.h>
using namespace std;
int main()
{
    int T;
    cin>>T;
    while(T--)
    {
        double a,b;
        scanf("%lf %lf",&a,&b);
        double ans=0;
        double A=a,B=b,temp=0;
        while(true)
        {
            if(A<0)
                temp=-pow(-A,1.0/3);
            else
                temp=pow(A,1.0/3);
            if(fabs(A-B)-fabs(temp-B)>1)
            {
                ans++;
                A=temp;
            }
            else
            {
                ans=ans+fabs(A-B);
                break;
            }
        }
        printf("%.9lf\n",ans);
    }
    return 0;
}

E.Enjoy the game(博弈)

https://ac.nowcoder.com/acm/contest/3006/E

题目描述

  初始一共有 $n$ 张卡牌，先手第一步最少要拿 $1$ 张牌，最多要拿 $n-1$ 张牌 $(2\leq n\leq 10^{18})$ 。接下来每一步，双方最少要拿 $1$ 张牌，最多拿等同于上一步对方拿的牌数的牌。拿走最后一张牌的人将取得游戏的胜利。 如果先手有必胜策略，输出Bob，否则输出Alice。

分析

  当张数是 $2^n$ 的时候输出Alice，否则输出Bob。

  如果不是 $2^n$ ，先手直接拿等同于其二进制最低位的数量的张数，后手将无力还击。

代码

#include<bits/stdc++.h>
using namespace std;
int main()
{
    long long n;
    cin>>n;
    int ans=__builtin_ctzll(n); 
    if((long long)((long long)1<<(long long)ans)==n)
        puts("Alice");
    else
        puts("Bob");
    return 0;
}

F.碎碎念(dp)

https://ac.nowcoder.com/acm/contest/3006/F

分析

考虑到了第 $i$ 句话时候，这种状态可能由两种状态转移而来。

  $1.$ 直接一发AC，从第 $i-1$ 句话直接到达第 $i$ 句。

  $2.$ 如果 $i>=k$，可以由一发 $RJ$ 从 $i-k$ 句到达第 $i$ 句。 构造 $dp[i][0/1]$，$dp[i][0]$ 表示到了第 $i$ 句话，是从 $i-1$ 句话AC到达的方案数。$dp[i][1]$ 表示到了第 $i$ 句话，是由 $i-k$ 句话 $RJ$ 过来的方案数。 由此可知转移条件：

  $dp[i][0]=dp[i−1][0]+dp[i−1][1]$ （前面一发是WA是AC无关紧要）

  $dp[i][1]=dp[i−k][0](i\geq k)$ (前面一发只能是AC)

  易确定边界条件 $dp[0][0]=1,dp[0][1]=0$。

  由于有 $q$ 次查询 ，查询数目较多，我们可以线性时间内预处理前缀和，然后用第 $r$ 项减去第 $l-1$ 项即可。

代码

#include <bits/stdc++.h>
using namespace std;
int dp[200010][2],sum[200010];
const int mod=1e9+7;
int main()
{
    int x;
    cin>>x;
    dp[0][0]=1;dp[0][1]=0;
    for(int i=1;i<100010;i++)
    {
        if(i>=x)
            dp[i][1]=(dp[i-x][0]+dp[i][0])%mod;
        dp[i][0]=(dp[i-1][0]+dp[i][0])%mod;
        dp[i][0]=(dp[i-1][1]+dp[i][0])%mod;
        sum[i]=(dp[i][0]+dp[i][1])%mod;
        sum[i]=(sum[i]+sum[i-1])%mod;
    }
    int q;
    cin>>q;
    while(q--)
    {
        int l,r;
        scanf("%d%d",&l,&r);
        printf("%d\n",(sum[r]-sum[l-1]+mod)%mod);
    }    
    return 0;
}

代码

#include <bits/stdc++.h>
using namespace std;
int dp[200010][2],sum[200010];
const int mod=1e9+7;
int main()
{
    int x;
    while(cin>>x)
    {
        memset(sum,0,sizeof(sum));
        dp[0][0]=1;
        for(int i=1;i<100010;i++)
        {
            if(i>=x)
                dp[i][1]=(dp[i-x][0]+dp[i][0])%mod;
            dp[i][0]=(dp[i-1][0]+dp[i][0])%mod;
            dp[i][0]=(dp[i-1][1]+dp[i][0])%mod;
            sum[i]=(dp[i][0]+dp[i][1])%mod;
            sum[i]=(sum[i]+sum[i-1])%mod;
        }
        int q;
        cin>>q;
        while(q--)
        {
            int l,r;
            scanf("%d%d",&l,&r);
            printf("%d\n",(sum[r]-sum[l-1]+mod)%mod);
        }   
    }
    return 0;
}

H.hash(思维)

https://ac.nowcoder.com/acm/contest/3006/H

题目描述

这里有一个 $hash$ 函数

const int LEN = 6;
int mod;
int Hash(char str[])
{
    int res=0;
    for(int i=0;i<LEN;i++)
        res=(res*26+str[i]-'a')%mod; 
    return res;
}

  现给定一个长度为 $6$ 的仅由小写字母构成的字符串 $s$ 和模数 $mod$ ，请找到字典序最小且大于 $s$ 的一个长度为 $6$ 的仅由小写字母构成的字符串 $s’$，使得其 $hash$ 值和 $s$ 的 $hash$ 相等。 $(2\leq mod \leq 10^7)$

分析

  只要把给的字符串转换成对应的 $26$ 进制整数，加上模数后转换回字符串，一定可以得到一个最小字典序大于原串的字符串。只要这个新字符串长度为 $6$ 即是满足要求的字符串。

代码

#include <bits/stdc++.h>
using namespace std;
int mod;
char s[15],t[15];
long long Hash(char str[]) 
{     
    long long res=0;     
    for(int i=0;i<6;i++)              
        res=res*26+str[i]-'a';      
    return res; 
}
int main()
{
    while(~scanf("%s %d",s,&mod))
    {
        long long temp=Hash(s)+mod;
        for(int i=0;i<6;i++)
        {
            t[6-i-1]=temp%26+'a';
            temp=temp/26;
        }
        if(strcmp(t,s)>0&&Hash(t)==Hash(s)+mod)
            cout<<t<<endl;
        else
            puts("-1");
    }
    return 0;
}

I.I题是个签到题(模拟)

https://ac.nowcoder.com/acm/contest/3006/I

题目描述

  通过人数大于等于全场人数的 $80%$ 或者通过人数是所有题目前三多的题叫做签到题。已知现在每个题的过题情况，判断 $I$ 题是不是一个签到题。

  输入数据共 $2$ 行。第一行包括两个整数 $n,m(9 \leq n \leq 13, 100 \leq m \leq 1,000)$，表示比赛的总题数和比赛的总人数。第二行包括以空格分隔的 $n$ 个整数 $a_1,a_2,\cdots,a_n(a_i > 0)$，表示每个题通过的人数。

代码

#include<bits/stdc++.h>
using namespace std;
int a[15];
int main()
{
    int n,m;
    cin>>n>>m;
    for(int i=1;i<=n;i++)
        cin>>a[i];
    if(a[9]>=0.8*m)
        puts("Yes");
    else
    {
        int ans=0;
        for(int i=1;i<=n;i++)
            if(a[i]>a[9])
                ans++;
        if(ans>2)
            puts("No");
        else
            puts("Yes");
    }
    return 0;
}

J.牛牛战队的秀场(简单几何)

题目描述

  牛牛和牛能来到一个半径为 $r$ 的圆上，画了圆内接的正 $n$ 边形。为了秀走位，他们只允许自己在多边形的边上移动。

  同时，他们随便选取正 $n$ 边形的一个顶点为 $1$ 号顶点，按顺时针的顺序把其他的点叫做 $2$ 号顶点，3号顶点……一开始，两人分别在 $i$ 号顶点和 $j$ 号顶点。

  现在，牛牛要一边沿着多边形的边秀走位，一边走向牛能。他想知道，他最短要走多少距离才能走到牛能的旁边？ $(3\leq n\leq 1000,1\leq r \leq 10^6,1\leq i,j\leq n)$

分析

  先求一条边的边长。根据余弦定理：$L^2=r^2+r^2-2·r·r·\cos \frac{2\pi}{n}$。$L=r\sqrt{2(1-\cos\frac{2\pi}{n})}$。

  再求 $i$ 到 $j$ 最短经过几条边：$\min (i-j,j-i+n)$。

代码

#include<iostream>
#include<algorithm>
using namespace std;
const double PI=acos(-1.0);
int main()
{
    double n,r;
    int A,B;
    cin>>n>>r;
    cin>>A>>B;
    if(A<B)  swap(A,B);
    double len=r*sqrt(2*(1-cos(2*PI/n)));
    printf("%.8lf\n",min((A-B)*len,(B+n-A)*len));
    return 0;
}
