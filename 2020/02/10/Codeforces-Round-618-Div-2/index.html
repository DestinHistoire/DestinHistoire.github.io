CF1300A Non-zero(水)

https://codeforces.com/contest/1300/problem/A

题目描述

  给定一个含有 $n$ 个整数的数列。每次可以将其中的一个数加 $1$，记为一次操作。求使数列所有数之和与数列所有数的乘积都不为 $0$ 的最少操作次数。$(1\leq n\leq 100,-100\leq a_i\leq 100)$

分析

  遇到数字 $0$ 操作次数需要加一，保证所有数的乘积不为 $0$，如果输入结束后序列的和为 $0$，操作次数需要再加一。

代码

#include<bits/stdc++.h>
using namespace std;
int main()
{
    int T;
    cin>>T;
    while(T--)
    {
        int n,ans=0,sum=0;
        cin>>n;
        for(int i=1;i<=n;i++)
        {
            int x;
            scanf("%d",&x);
            sum=sum+x;
            if(x==0)
                sum++,ans++;
        }
        if(sum==0)
            ans++;
        cout<<ans<<endl;
    }
    return 0;
}

CF1300B Assigning to Classes(水)

https://codeforces.com/contest/1300/problem/B

题目描述

  给定一个 $2\times n$ 个数的数列。把它分成两组，使得每一组的数量均为奇数。求两组中位数之差的绝对值的最小值。$(1\leq n\leq 10^5,1\leq a_i\leq 10^9)$

分析

  答案为排序后的两个中位数之差。

代码

#include<bits/stdc++.h>
using namespace std;
int a[200010];
int main()
{
    int T;
    cin>>T;
    while(T--)
    {
        int n;
        cin>>n;
        for(int i=1;i<=2*n;i++)
            scanf("%d",&a[i]);
        sort(a+1,a+1+2*n);
        cout<<a[n+1]-a[n]<<endl;
    }
    return 0;
}

CF1300C Anu Has a Function(位运算)

https://codeforces.com/contest/1300/problem/C

题目描述

  给定一个 $n$ 个数的序列，重新排列此序列，使 $f(f(…f(f(a_1,a_2),a_3),…,a_{n−1}),a_n)$ 最大。

其中 $f(x,y)=(x|y)-y$。$(1\leq n\leq 10^5.0\leq a_i\leq 10^9)$

分析

  $y$ 的某一位为 $0$ 的话 $x$ 的这一位不变，$y$ 的某一位为 $1$ 的话 $x$ 的这一位为 $0$，因此 $f(x,y)=(x|y)-y=x&(_\tilde\ y)$。

  $f(f(…f(f(a_1,a_2),a_3),…,a_{n−1}),a_n)=a_1&(_\tilde\ a_2)&(_\tilde\ a_3)&…&(_\tilde\ a_n)=a_1&{_\tilde\ (a_2|a_3|…|a_n)}$ 因此只需要找到某一个数字 $a_k$ 与 其余数字的或和最大即可。

  $a_k&{\tilde\ (a_1|a_2|…|a{k-1})}&{\tilde\ (a{k+1}|a_{k+2}|…|a_n)}=a_1|a_2|…|a_n-a_1|a_{2}|…|a_{k-1}|a_{k+1}|…|a_n)}$

  预处理序列的或和前缀 $dp1[i]$ 和序列的或和后缀 $dp2[i]$ ，记录令函数值最大的 $k$ 后输出 $a_k$，其余数字依次输出即可。时间复杂度 $O(n)$。

代码

#include<bits/stdc++.h>
using namespace std;
int a[200010],dp1[200010],dp2[200010];
int main()
{
    int n;
    cin>>n;
    for(int i=1;i<=n;i++)
        scanf("%d",&a[i]);
    for(int i=1;i<=n;i++)
        dp1[i]=dp1[i-1]|a[i];
    for(int i=n;i>=1;i--)
        dp2[i]=dp2[i+1]|a[i];
    int ans=-1,pos=-1;
    for(int i=1;i<=n;i++)
    {
        int temp=(a[i]|dp1[i-1]|dp2[i+1])-(dp1[i-1]|dp2[i+1]);
        if(temp>ans)
        {
            pos=i;
            ans=temp;
        }
    }
    cout<<a[pos]<<" ";
    for(int i=1;i<=n;i++)
    {
        if(i==pos)
            continue;
        cout<<a[i]<<" ";
    }
    cout<<endl;
    return 0;
}

CF1300D Aerodynamic(几何)

https://codeforces.com/contest/1300/problem/D

题目描述

  给定 $n$ 个点构成的凸包 $P$，在原点 $(0,0)$ 在此凸包的范围内的前提下，任意平移该凸包，组成的图形的交集称为 $T$，判断 $T$ 和 $P$ 是否相似。$(3\leq n\leq 10^5,|x_i|,|y_i|\leq 10^9)$

分析

  判断该图形是否为中心对称图形即可。

代码

#include<bits/stdc++.h>
using namespace std;
int n,x[200010],y[200010];
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
        scanf("%d%d",&x[i],&y[i]);
    if(n&1)
    {
        puts("NO");
        return 0;
    }
    for(int i=1;i<=n/2;i++)
    {
        if(x[i]+x[i+n/2]!=x[1]+x[1+n/2]||y[i]+y[i+n/2]!=y[1]+y[1+n/2])
        {
            puts("NO");
            return 0;
        }
    }
    puts("YES");
    return 0;
}

CF1300E Water Balance(贪心)

题目描述

  给定一个序列，每次操作区间 $[l,r]$ ，使区间内的所有数字变为平均值，可操作任意次，求可能获得的字典序最小的序列。$(1\leq n\leq 10^6,1\leq a_i\leq 10^6)$

分析

  首先把每个点看成一个区间，然后倒着遍历，如果这个区间的平均值比前一个区间平均值小，就将这个区间和前一个区间合并。

代码

#include<bits/stdc++.h>
using namespace std;
const int SIZE=1e6+10;
double a[SIZE],len[SIZE],st[SIZE];
int main()
{
    int n,cnt=0;
    cin>>n;
    for(int i=1;i<=n;i++)
        scanf("%lf",&a[i]);
    for(int i=1;i<=n;i++)
    {
        st[++cnt]=a[i];
        len[cnt]=1;
        while(cnt>1&&st[cnt]<st[cnt-1])
        {
            st[cnt-1]=(st[cnt]*len[cnt]+st[cnt-1]*len[cnt-1])/(len[cnt-1]+len[cnt]);
            len[cnt-1]=len[cnt-1]+len[cnt];
            cnt--;
        }
    }
    for(int i=1;i<=cnt;i++)
        for(int j=1;j<=len[i];j++)
            printf("%.8lf\n",st[i]);
    cout<<endl;
}
