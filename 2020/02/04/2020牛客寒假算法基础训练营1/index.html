A.honoka和格点三角形(计数)

https://ac.nowcoder.com/acm/contest/3002/A

题目描述

honoka 最近在研究三角形计数问题。
她认为，满足以下三个条件的三角形是“好三角形”。
$1.$三角形的三个顶点均为格点，即横坐标和纵坐标均为整数。
$2.$三角形的面积为 $1$。
$3.$三角形至少有一条边和 $x$ 轴或 $y$ 轴平行。
honoka想知道，在平面中选取一个大小为 $n*m$ 的矩形格点阵，可以找到多少个不同的“好三角形”？由于答案可能过大，请对 $1000000007$ 取模。$(1\leq n,m\leq 10^9)$

分析

$1.$ 平行于 $x$ 轴且边长度为 $2$，高为 $1$ 的三角形种类：$2(m-2)*m(n-1)$

$2.$ 平行于 $y$ 轴且边长度为 $2$，高为 $1$ 的三角形种类：$2(n-2)*n(m-1)$

$3.$ 平行于 $x$ 轴且边长度为 $1$，高为 $2$ 的三角形种类：$2(m-1)(m-2)*(n-2)$（去掉长和高都与 $x,y$ 轴平行的情况）

$4.$ 平行于 $y$ 轴且边长度为 $1$，高为 $2$ 的三角形种类：$2(n-1)(n-2)*(m-2)$ （去掉长和高都与 $x,y$ 轴平行的情况）

代码

#include<bits/stdc++.h>
using namespace std;
const int mod=1e9+7;
int main()
{
    long long n,m;
    cin>>n>>m;
    long long ans=0;
    ans=ans+2*(m-2)*m%mod*(n-1)%mod;
    ans=ans+2*(n-2)*n%mod*(m-1)%mod;
    ans=ans+2*(n-1)*(n-2)%mod*(m-2)%mod;
    ans=ans+2*(m-1)*(m-2)%mod*(n-2)%mod;
    cout<<ans%mod<<endl;
    return 0;
}

B.kotori和bangdream(概率)

https://ac.nowcoder.com/acm/contest/3002/B

题目描述

  每个音符 $x%$ 的概率perfect，获得 $a$ 分， $(100−x)%$ 概率great，获得 $b$ 分。已知一首歌有 $n$ 个音符。求一首歌得分的期望是多少？$(0\leq x\leq 100,1\leq n,a,b \leq 1000)$

分析

  答案为 $n(a*x+b(100-x))/100)$

代码

#include <bits/stdc++.h>
using namespace std;
int main()
{
    int n,x,a,b;
    cin>>n>>x>>a>>b;
    printf("%.2lf\n",1.0*n*(a*x+b*(100-x))/100);    
}

C.umi和弓道(计算几何+双指针)

https://ac.nowcoder.com/acm/contest/3002/C

题目描述

umi对弓道非常痴迷。
有一天，她在研究一个射箭问题：
在一个无限大的平面中，她站在 $(x_0,y_0)$ 这个坐标。
有 $n$ 个靶子，第 $i$ 个靶子的坐标是 $(x_i,y_i)$。
umi 准备在 $x$ 轴或 $y$ 轴上放置一块挡板来挡住弓箭的轨迹，使得她可以射中的靶子数量不超过 $k$ 个。
她想知道挡板的最短长度是多少？
注：假定弓箭的轨迹是起点为 umi 坐标、长度无穷大的射线。umi 和靶子的体积可以无视。挡板的边缘碰到弓箭轨迹也可视为挡住弓箭。
注2：挡板不能弯折，起始和终点必须在同一坐标轴上。

$(1\leq n \leq 100000,0\leq k\leq n-2,-210^9\leq x_i,y_i\leq 210^9)$

输入描述

第一行两个整数 $x_0,y_0$，代表 umi的坐标。

第二行两个正整数 $n$ 和 $k$，分别代表靶子的总数量、放置挡板后可射中靶子的最大值。

接下来的 $n$ 行，每行两个整数 $x_i$ 和 $y_i$。代表每个靶子的坐标。保证没有任何一个点在坐标轴上（无论umi还是靶子），保证没有任何两点重合。

输出描述

若无论如何无法保证可以射中的靶子数量不超过 $k$ 个，则输出 $-1$。

否则输出挡板的最小长度。如果你和正确答案的误差不超过 $10^{-6}$，则视为答案正确。

分析

  首先确定umi所在位置的象限。很明显同一象限的点是不可能用挡板挡掉的，对于剩下的点找出线段和 $x$ 轴或 $y$ 轴的交点，统计坐标位置（如果存在交点的话）。
  之后双指针维护 $x$ 轴上或者 $y$ 轴挡住 $n−k$ 个点的挡板长度最小值。$x$ 轴和 $y$ 轴要分开计算。

代码

#include<bits/stdc++.h>
using namespace std;
const int INF=0x3f3f3f3f;
int n,k;
double x[100010],y[100010];
double dist1[100010],dist2[100010];
int main()
{
    cin>>x[0]>>y[0];
    cin>>n>>k;
    for(int i=1;i<=n;i++)
        scanf("%lf%lf",&x[i],&y[i]);
    int cnt1=0,cnt2=0;
    for(int i=1;i<=n;i++)
    {
        if(x[0]*x[i]<0)
            dist1[++cnt1]=y[0]-x[0]*(y[i]-y[0])/(x[i]-x[0]);    
        if(y[0]*y[i]<0)
            dist2[++cnt2]=x[0]-y[0]*(x[i]-x[0])/(y[i]-y[0]);
    }
    double ans=INF;
    sort(dist1+1,dist1+cnt1+1);
    sort(dist2+1,dist2+cnt2+1);
    k=n-k;
    if(cnt1>=k)
    {
        int l=1,r=k;
        while(r<=cnt1)
        {
            ans=min(ans,dist1[r]-dist1[l]);
            r++,l++;
        }
    }
    if(cnt2>=k)
    {
        int l=1,r=k;
        while(r<=cnt2)
        {
            ans=min(ans,dist2[r]-dist2[l]);
            r++,l++;
        }
    }
    if(ans==INF)
        cout<<-1<<endl;
    else
        printf("%.7lf\n",ans);
    return 0;
}

D.hanayo和米饭(水)

https://ac.nowcoder.com/acm/contest/3002/D

题目描述

有 $n$ 个碗，第一个碗装了 $n$ 粒米饭，第二个碗装了 $2$ 粒米饭，以此类推，第 $n$ 个碗装了 $n$ 粒米饭。
然而，爱搞恶作剧的rin把所有的碗的顺序打乱，并拿走了一个碗。hanayo想知道，rin拿走的碗里有多少粒米饭？$(2\leq n \leq 100000,1\leq a_i\leq n)$

分析

自然数前 $n$ 项和为 $\frac{n(n+1)}{2}$，减去序列每一项就是答案。

代码

#include <bits/stdc++.h>
using namespace std;
int a[100010];
int main()
{
    int n;
    cin>>n;
    long long sum=(long long)n*(n+1)/2;
    for(int i=1;i<=n-1;i++)
    {
        scanf("%d",&a[i]);
        sum=sum-a[i];
    }
    cout<<sum<<endl;
    return 0;
}

E.rin和快速迭代(数学)

https://ac.nowcoder.com/acm/contest/3002/E

题目描述

设 $f(x)$ 为 $x$ 的因子个数，将 $f$ 迭代下去，rin猜想任意正整数最终都会变成 $2$ 。
例如：$f(12)=6,f(6)=4,f(4)=3,f(3)=2$。
给一个正整数 $n$，输出它在迭代过程中，第一次迭代成 $2$ 的迭代次数。$(3\leq n\leq 10^{12})$

分析

$O(\sqrt{n})$ 复杂度处理每一次迭代。

代码

#include<bits/stdc++.h>
using namespace std;
long long f(long long x)//求x的因子个数 
{
    long long i,ans=0;
    for(i=1;i*i<x;i++)
        if(x%i==0)
            ans=ans+2;
    if(i*i==x)
        ans++;
    return ans;
}
int main()
{
    long long n,cnt=0;
    cin>>n;
    while(n!=2)
    {
        n=f(n);
        cnt++;
    }
    cout<<cnt<<endl;
}

F.maki和tree(并查集)

https://ac.nowcoder.com/acm/contest/3002/F

题目描述

有 $n$ 个点，$n-1$ 条边的树。每个顶点被染成了白色或者黑色。
取两个不同的点，它们的简单路径上有且仅有一个黑色点的取法有多少种？
注：
①树上两点简单路径指连接两点的最短路。
②$(x,y)$ 和 $(y,x)$ 的取法视为同一种。

输入描述

第一行一个正整数 $n$ 。代表顶点数量。$(1\leq n\leq 100000)$

第二行是一个仅由字符 $’B’$ 和 $’W’$ 组成的字符串。第 $i$ 个字符是 $B$ 代表第 $i$ 个点是黑色，$W$ 代表第 $i$ 个点是白色。

接下来的 $n-1$ 行，每行两个正整数 $x,y$ ，代表 $x$ 点和 $y$ 点有一条边相连。$(1\leq x,y \leq n)$

输出描述

一个正整数，表示只经过一个黑色点的路径数量。

分析

经过一个黑点的路径有两种：两个端点都是白点；其中一个端点是黑点。

可以先预处理，将每个白点连通块上的白点个数统计出来，这样可以得知每个黑点所连接的白点的权值（即连通块白点数）。

设某黑点连接了 $k$ 个白点，第 $i$ 个白点的权值为 $f(i)$。

这样第一种路径的数量为 $\displaystyle\sum\limits_{i=1}^{k}\displaystyle\sum\limits_{j=i+1}^{k}f(i)*f(j)$，第二种路径的数量为 $\displaystyle\sum\limits_{i=1}^{k}f(i)$。第一种用前缀和处理达成 $O(k)$ 计算。

代码

#include<bits/stdc++.h>
using namespace std;
const int SIZE=100010;
int fa[SIZE],siz[SIZE],num[SIZE];//siz为儿子的数量,num为白点连通块上的白点个数 
vector<int> G[SIZE],temp;
char str[SIZE];
long long sum[SIZE];
int get(int x)
{
    if(x==fa[x])
        return x;
    return fa[x]=get(fa[x]);
}
void merge(int x,int y)
{
    int fx=get(x),fy=get(y);
    if(fx!=fy)
    {
        if(siz[fx]>siz[fy])
        {
            fa[fy]=fx;
            siz[fx]=siz[fx]+siz[fy]+1;
        }
        else
        {
            fa[fx]=fy;
            siz[fy]=siz[fy]+siz[fx]+1;
        }
    }
}
long long solve()
{
    if(temp.size()==0)
        return 0;
    int n=temp.size();
    sum[n]=0;
    for(int i=n-1;i>=0;i--)
        sum[i]=sum[i+1]+temp[i];
    long long res=sum[0];
    for(int i=0;i<=n-1;i++)
        res=res+temp[i]*sum[i+1];
    return res;
}
int main()
{
    int n,x,y;
    cin>>n;
    scanf("%s",str+1);
    for(int i=1;i<=n;i++)
        fa[i]=i;
    for(int i=1;i<=n-1;i++)
    {
        scanf("%d%d",&x,&y);
        G[x].push_back(y);
        G[y].push_back(x);
        if(str[x]=='W'&&str[y]=='W')
            merge(x,y); 
    }
    for(int i=1;i<=n;i++)
        num[i]=siz[fa[i]]+1;
    long long ans=0;
    for(int i=1;i<=n;i++)
    {
        if(str[i]=='B')
        {
            for(int j=0;j<G[i].size();j++)
                if(str[G[i][j]]=='W')
                    temp.push_back(num[G[i][j]]);
            ans=ans+solve();
            temp.clear();
        }
    }
    cout<<ans<<endl;
    return 0;
}

G.eli和字符串(尺取法)

https://ac.nowcoder.com/acm/contest/3002/G

题目描述

  给定一个仅由小写字母组成的字符串。截取一段连续子串，这个子串包含至少 $k$ 个相同的某个字母。求子串的长度最小值是多少？如果无论怎么取都无法满足条件，输出 $-1$。否则输出一个正整数，为满足条件的子串长度最小值。$(1\leq k\leq n\leq 200000)$

分析

  先初始化左右端点为 $l=0,r=-1$ 。然后找到右端点 $r$，使得区间 $[l,r]$ 合法。移动左端点 $l$，使得区间 $[++l,r]$合法。不停的移动左右区间。

代码

#include <bits/stdc++.h>
using namespace std;
const int INF=0x3f3f3f3f;
char str[200010];
int book[30];
int n,k;
bool check()
{
    for(int i=0;i<26;i++)
        if(book[i]>=k)
            return true;
    return false;
}
int main()
{
    cin>>n>>k;
    cin>>str;
    int l=0,r=-1,ans=INF;
    for(int i=0;i<n;i++)
    {
        r++;
        book[str[r]-'a']++;
        while(check())
        {
            ans=min(ans,r-l+1);
            book[str[l]-'a']--;
            l++;
        }
    }
    if(ans==INF)
        cout<<-1<<endl;
    else
        cout<<ans<<endl;
    return 0;
}

H.nozomi和字符串(贪心)

https://ac.nowcoder.com/acm/contest/3002/H

题目描述

对于一个 $“01”$ 串而言，每次操作可以把 $0$ 字符改为 $1$ 字符，或者把 $1$ 字符改为 $0$ 字符。所谓 $“01”$ 串，即只含字符 $0$ 和字符 $1$ 的字符串。有最多 $k$ 次操作的机会。在操作之后找出一个尽可能长的连续子串，这个子串上的所有字符都相同。求子串的长度最大值是多少？
注：$k$ 次操作机会可以不全部用完。$(1\leq k\leq n\leq 200000)$

分析

  显然操作要么全 $1$ 变 $0$ 要么全 $0$ 变 $1$。

  分别处理两种操作即可。对于 $1$ 变 $0$ 的情况，分别统计每个 $1$ 的位置（假设第一个 $1$ 的位置为 $-1$，最后一个 $1$ 的位置为 $n$），那么 $k$ 次操作，即变换连续 $k$ 个 $1$，最终的字符串长度为第 $i$ 个 $1$ 的位置到第 $i+k$ 个 $1$ 的位置之间的距离。

  对于 $0$ 变 $1$ 的情况同理。

代码

#include<bits/stdc++.h>
using namespace std;
char str[200010];
int pos0[200010],pos1[200010];
int cnt0,cnt1;
int main()
{
    int n,k;
    cin>>n>>k;
    cin>>str;
    cnt0=cnt1=0;
    pos0[0]=-1;
    pos1[0]=-1;
    for(int i=0;i<n;i++)
    {
        if(str[i]=='0')
            pos0[++cnt0]=i;
        else
            pos1[++cnt1]=i;
    }
    pos0[++cnt0]=n;
    pos1[++cnt1]=n;
    int len0=cnt0+1,len1=cnt1+1,ans=0;
    if(len0-2<=k)
        ans=n;
    else
    {
        for(int i=1,j=k;j<=len0-2;i++,j++)
            ans=max(ans,pos0[j+1]-pos0[i-1]-1);
    }
    if(len1-2<=k)
        ans=n;
    else
    {
        for(int i=1,j=k;j<=len1-2;i++,j++)
            ans=max(ans,pos1[j+1]-pos1[i-1]-1);
    }
    cout<<ans<<endl;
    return 0;
}

I.nico和niconiconi(dp)

https://ac.nowcoder.com/acm/contest/3002/I

题目描述

“nico” 计 $a$ 分，”niconi” 计 $b$ 分，”niconiconi” 计 $c$ 分。有一个长度为 $n$ 的字符串，请帮她算出最大计数分数。
注：已被计数过的字符不能重复计数！如”niconico”要么当作”nico”+”nico”计 $2a$ 分，要么当作”niconi”+”co”计 $b$ 分。$(1\leq n\leq 300000,1\leq a,b,c\leq 10^9)$

分析

设 $dp[i]$ 代表前 $i$ 个字符的计数最大值。
转移方程：

$if(substring(i−3,i)==nico)$ $dp[i]=max(dp[i],dp[i−4]+a)$
$if(substring(i−5,i)==niconi)$ $dp[i]=max(dp[i],dp[i−6]+b)$
$if(substring(i−9,i)==niconiconi)$ $dp[i]=max(dp[i],dp[i−10]+c)$

目标：$dp[n]$。

代码

#include<bits/stdc++.h>
using namespace std;
long long dp[300010];
char str[300010];
int main()
{
    int n,a,b,c;
    cin>>n>>a>>b>>c;
    cin>>str;
    for(int i=0;i<n;i++)
    {
        if(i>0)
            dp[i]=dp[i-1];
        if(i>=3&&str[i-3]=='n'&&str[i-2]=='i'&&str[i-1]=='c'&&str[i]=='o')
            dp[i]=max(dp[i],dp[i-3]+a);
        if(i>=5&&str[i-5]=='n'&&str[i-4]=='i'&&str[i-3]=='c'&&str[i-2]=='o'&&str[i-1]=='n'&&str[i]=='i')
            dp[i]=max(dp[i],dp[i-5]+b);
        if(i>=9&&str[i-9]=='n'&&str[i-8]=='i'&&str[i-7]=='c'&&str[i-6]=='o'&&str[i-5]=='n'&&str[i-4]=='i'&&str[i-3]=='c'&&str[i-2]=='o'&&str[i-1]=='n'&&str[i]=='i')
            dp[i]=max(dp[i],dp[i-9]+c);
    }
    cout<<dp[n-1]<<endl;//，对于字符str[i]='n'的情况，显然dp[i]=dp[i-1]。这样就不用处理dp[-1]的值。
    return 0;
}

J.u’s的影响力(矩阵快速幂+费马小定理)

https://ac.nowcoder.com/acm/contest/3002/J

题目描述

$f(1)=x,f(2)=y,f(i)=f(i-1)f(i-2)a^b(i>2)$

求 $f(n) \bmod 1000000007(1\leq x,x,y,a,b\leq 10^{12})$

分析

$fib(1)=1,fib(2)=1,fib(3)=2$

$f(3)=xya^b$

$f(4)=xy^2a^{2b}$

$f(5)=x^2y^3a^{4b}$

$……$

归纳可得：$f(n)=x^{fib(n-2)}y^{fib(n-1)}a^{fib(n)-1}$

根据费马小定理，$a^{1e9+6}\equiv 1\pmod{1e9+7}$。因此幂的指数对 $1e9+6$ 取模即可，逐一 $a$ 是 $1e9+7$ 的倍数的特殊情况。

假设 $f(i)$ 代表斐波那契数列的第 $i$ 项，则有：

$\left[ \begin{matrix} 0 & 1 \\ 1 & 1 \end{matrix} \right]*\left[ \begin{matrix} f(i) & 0\\ f(i+1) & 0 \end{matrix} \right]=\left[ \begin{matrix} f(i+1) & 0\\ f(i)+f(i+1) & 0 \end{matrix} \right]=\left[ \begin{matrix} f(i+1) & 0\\ f(i+2) & 0 \end{matrix} \right]$

因此 $\left[ \begin{matrix} 0 & 1 \\ 1 & 1 \end{matrix} \right]^n*\left[ \begin{matrix} f(1) & 0 \\ f(2) & 0 \end{matrix} \right]=\left[ \begin{matrix} f(n+1) & 0 \\ f(n+2) & 0 \end{matrix} \right]$

代码

#include <bits/stdc++.h>
using namespace std;
const int mod=1e9+7;
const int p=1e9+6;
struct matrix
{
    long long a[2][2];
};
matrix mul(matrix A,matrix B)
{
    matrix ans;
    for(int i=0;i<2;i++)
        for(int j=0;j<2;j++)
            ans.a[i][j]=0;
    for(int i=0;i<2;i++)
        for(int j=0;j<2;j++)
            for(int k=0;k<2;k++)
                ans.a[i][j]=(ans.a[i][j]+(A.a[i][k]*B.a[k][j])%p)%p;
    return ans;    
}
long long fib(long long n)
{
    n--;
    matrix A,ans;
    A.a[0][0]=0;A.a[0][1]=1;
    A.a[1][0]=1;A.a[1][1]=1;

    ans.a[0][0]=1;ans.a[0][1]=0;
    ans.a[1][0]=1;ans.a[1][1]=0;
    while(n)
    {
        if(n&1)
            ans=mul(A,ans);
        A=mul(A,A);
        n>>=1;
    }
    return ans.a[0][0]%p;
}
long long quick_pow(long long a,long long b,long long c)
{
    long long ans=1;
    long long base=a%c;
    while(b)
    {
        if(b&1)
            ans=ans*base%c;
        base=base*base%c;
        b>>=1;
    }
    return ans;
}
