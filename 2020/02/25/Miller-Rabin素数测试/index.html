Miller-Rabin素数测试
数论 米勒拉宾素数测试
算法笔记
发布日期:   2020-02-25

同余的性质

$1.$ 同加性：若 $a\equiv b\pmod n$，则 $a+c\equiv b+c\pmod n$。

$2.$ 同乘性：若 $a\equiv b\pmod n$，则 $a\ast c\equiv b\ast c\pmod n$。

     若 $a\equiv b\pmod n,c\equiv d\pmod n$，则 $a\ast c\equiv b\ast d\pmod n$。

$3.$ 若 $\gcd(c,n)=1$，且 $a\ast c\equiv b\ast c \pmod n$ ，则 $a\equiv b\pmod n$，即同余式两边可以同时除以一个与模数互质的数。

证明：

  已知 $a\ast c+k_1\ast n=b\ast c+k_2*n$，则 $(a-b)\ast c=n\ast (k_2-k_1)$。这表明 $n\mid(a-b)\ast c$ ，由于 $\gcd(c,n)=1$，因此只有可能是 $n\mid(a-b)$，即 $a\equiv b\pmod n$。

Wilson定理

  若 $p$ 为质数，则 $(p-1)!\equiv -1\pmod p$

证明：

  当 $p=2$ 时，有 $(p-1)!\equiv 1\equiv-1\pmod 2$。因此，当 $p=2$ 时定理成立。现在设 $p$ 是大于 $2$ 的质数。对于每个满足 $1\leq a\leq p-1$ 的整数 $a$，存在逆 $\overline{a}$，使得 $1\leq \overline{a}\leq p-1$ 且 $a\overline{a}\equiv 1 \pmod p$。由于在小于 $p$ 的正整数中，逆是其本身的数只有 $1$ 和 $p-1$。因此，可以将 $2$ 到 $p-2$ 分成 $\frac{(p-3)}{2}$ 组整数对，并且每组的乘积模 $p$ 余 $1$。
$$
2·3\cdots(p-3)·(p-2)\equiv 1\pmod p
$$
  将上式两边同时乘以 $1$ 和 $p-1$ ，得
$$
(p-1)!=1·2·3\cdots(p-3)·(p-2)·(p-1)=1·(p-1)\equiv-1\pmod p
$$
得证。

  威尔逊定理的逆定理也成立：对于某一正整数 $n$，若 $(n-1)!\equiv-1\pmod n$ ，则 $n$ 是质数。

证明：

  假设 $n$ 是一个合数并且 $(n-1)!\equiv-1\pmod n$。因为 $n$ 是合数，有 $n=ab$，其中 $1<a<n$ 且 $1<b<n$。

  因为 $a<n$ ，则 $a$ 是组成 $(n-1)!$ 的 $n-1$ 个数中的一个，故 $a\mid(n-1)!$。因 $(n-1)!\equiv-1\pmod n$，故 $n\mid((n-1)!+1)$。这意味着，$a$ 也整除 $(n-1)!+1$。

  利用 $a\mid(n-1)!$ 且 $a\mid((n-1)!+1)$，知 $a\mid((n-1)!+1)-(n-1)!=1$，这与 $a>1$ 矛盾，因此 $n$ 是一个质数。

费马小定理

  如果 $p$ 是一个质数，且 $0<a<p$ ，则 $a^{p-1}\equiv 1\pmod p$。

证明：

  $1\ast a%p=a_1$

  $2\ast a%p=a_2$

  $\cdots\cdots$

  $(p-1)\ast a% p=a_{p-1}$

易知，$i\neq j$ 时，$a_i\neq a_j$，则 $a_1,a_2,\cdots,a_{p-1}$ 为 $1,2,3,\cdots,p-1$ 的一个全排列。

将上式两边全部乘在一起，得到：$(p-1)!\ast a^{p-1}%p=(p-1)!$

即 $a^{p-1}\equiv 1\pmod p$，得证。

费马素性检验

  费马小定理逆命题：若 $a^p\equiv p\pmod p$，$p$ 为质数

  这个命题是错误的，但是错误概率极小。

  1819年有人发现了费马小定理逆命题的第一个反例：虽然 $2^{340}\ %\ 341=1$，但 $341=11\ast 31$。后来，人们又发现了 $561,645,1105$ 等数都表明 $a=2$ 时费马小定理的逆命题不成立。人们把所有能整除 $2^{(n-1)}-1$ 的合数 $n$ 叫做 伪素数（$\text{pseudoprime}$）。

  不满足 $2^{n-1}\equiv 1\pmod n$ 的 $n$ 一定不是质数；如果满足的话则多半是质数。这样，一个比试除法效率更高的素性判断方法出现了：制作一张伪素数表，记录某个范围内的所有伪素数，那么所有满足 $2^{n-1}\equiv1\pmod n$ 且不在伪素数表中的 $n$ 就是质数。之所以这种方法更快，是因为可以使用快速幂快速计算 $2^{n-1} \text{mod}\ n$ 的值。

  刚才只考虑了 $a=2$ 的情况。一个合数可能在 $a=2$ 时通过了测试，但 $a=3$ 时的计算结果却排除了素数的可能。于是，扩展伪素数的定义，称满足 $a^{n-1}\ \text{mod} \ n=1$ 的合数 $n$ 叫做以 a为底的伪素数（$\text{pseudoprime to base a}$）。

  随机选择若干个小于待测数的正整数作为底数 $a$ 进行若干次测试，只要有一次没有通过测试就立即把这个数判断为合数。这就是费马素性检验。

  费马小定理毕竟质数判定的一个必要条件。满足费马小定理条件的整数 $n$ 未必全是质数。对于合数 $n$，如果对于所有与 $n$ 互质的正整数 $a$，都有 $a^{n-1}\equiv 1\pmod n$ 成立，则称合数 $n$ 为卡迈克尔数。前 $3$ 个卡迈克尔数是 $561,1105,1729$。在 $1$ ~$10^8$ 范围内的整数中，只有 $255$ 个卡迈克尔数。

  费马小定理的前提是 $a$ 和 $n$ 互质。当 $n$ 本身就是质数的时候如果 $a<n$ 那么 $a$ 和 $n$ 始终互质；但 $n$ 不是素数时，如果 $a$ 和 $n$ 不互质的话不能用费马小定理。也就是说，卡迈克尔数需要排除 $a$ 和 $n$ 不互质的情况。

  利用下面的二次探测定理可以对上面的素数判定算法作进一步改进，以避免将卡迈克尔数当作质数。

Miller-Rabin素数测试

二次探测定理优化

新的测试基于下面的定理：

  如果 $p$ 是质数，$x<p$ 且 $x^2\equiv 1\pmod p$，解为 $x_1=1,x_2=p-1$。

证明：

  因为 $x^2\equiv 1\pmod p$，所以 $p\mid (x^2-1)$ ，即 $p\mid(x-1)\ast(x+1)$。

  由于 $p$ 是质数，那么只可能是 $p\mid (x-1)$ （此时 $x=1$）或 $p\mid (x+1)$（此时 $x=p-1$）。

演示

  $341$ 可以通过以 $2$ 为底的费马素性检验，因为 $2^{340}%341 =1$。如果 $341$ 的确是质数的话，那么 $2^{170}\ %\ 341$ 的结果只可能是 $1$ 或 $340$ ；当算得 $ 2^{170}\ %\ 341 $ 确实等于 $1$ 时，继续查看 $2^{85}\ %\ 341$ 的结果，发现，$2^{85}\ %\ 341=32$，这一结果判定 $341$ 是合数。

  这就是 $\text{Miller-Rabin}$ 素数测试的方法。不断地提取指数 $n-1$ 中的因子 $2$，把 $n-1$ 表示成 $d\ast 2^r$（$d$ 是一个奇数）。需要计算的东西就变成了 $a^{d\ast2^r}$ 模 $n$ 的余数。

  所以 $a^{d\ast 2^{r-1}}$ 要么等于 $1$，要么等于 $n-1$。如果 $a^{d\ast 2^{r-1}}=1$，定理继续适用于 $a^{d\ast 2^{r-2}}$，这样不断迭代下去，直到存在 $i$ 满足 $a^{d\ast2^{r-i}}%\ n=n-1$ ，或者最后指数中的 $2$ 消耗完得到 $a^d\ %\ n=1$ 或 $a^d %\ n=n-1$。

  这样，费马小定理加强为如下形式：

  尽可能提取因子 $2$，把 $n-1$ 表示成 $d\ast 2^r$ ，如果存在 $i$ 满足 $a^{d\ast 2^{r-i}} %\ n=n-1(0\leq i<r)$，或者 $a^d\ %\ n = 1 $，则 $n$ 是一个质数。

总结

  若 $n$ 通过一次测试，则 $n$ 不是素数的概率为 $25%$，若 $n$ 通过 $t$ 次测试，则 $n$ 不是素数的概率为$\frac{1}{4^t}$。事实上取 $t$ 为 $5$ 时，$n$ 不是素数的概率为 $\frac{1}{128}$，$n$ 为素数的概率已经大于 $99.99%$ 。在实际应用中，可首先用 $300$ ~ $500$ 个小素数对 $n$ 进行测试，以提高 $\text{Miller-Rabin}$ 素数测试通过的概率，从而提高测试速度。

  把可以通过以 $a$ 为底的 $\text{Miller-Rabin}$ 测试的合数称作以 $a$ 为底的强伪素数（$\text{strong pseudoprime}$）。第一个以 $2$ 为底的强伪素数为 $2047$ ，第一个以 $2$ 和 $3$ 为底的强伪素数为 $1373653$，以 $2,3,5,7$ 为底，在 $2.5\ast 10^{13}$ 以内唯一一个判断失误的数为 $3215031752$。

【例题】HOJ-1356 Prime Judge

http://acm.hit.edu.cn/problemset/1356

题目描述

  判断 $n(n\leq 2^{31})$ 是否为质数。

代码

#include<bits/stdc++.h>
using namespace std;
int prime[]={0,2,3,7,61,24251};
long long mul(long long a,long long b,long long p) 
{
    return a*b%p;
}
long long quick_pow(long long a,long long b,long long p) 
{
    long long ans=1;
    while(b)
    {
        if(b&1) 
            ans=mul(ans,a,p);
        a=mul(a,a,p);
        b>>=1;
    }
    return ans;
}
int TwiceDetect(long long a,long long b,long long p) 
{
    int t=0;
    long long x, y;
    while(b%2==0) 
    {
        b>>=1;
        t++;
    }
    //b=d*2^t;b=d;
    x=y=quick_pow(a,b,p);//x=y=a^d%n
    //二次探测定理是反向递归的,也可以用如下的正向迭代法去测试
    while(t--) 
    {
        y=mul(x,x,p);
        if(y==1&&x!=1&&x!=p-1)//注意y!=1的时候是不做判断的,即对应递归时在某一环节x==p-1的情况,对此x开方则无意义,
            return 0;
        x=y;///继续向高次迭代,那么至少最后一次应该是等于1的(费马小定理)
    }
    return y;
}
bool Miller_Rabin(long long n) 
{
    int i;
    for(i=1;i<=5;i++) 
    {
        if(n==prime[i]) 
            return true;
        if(TwiceDetect(prime[i],n-1,n)!=1)
            break;
    }
    return i==6;
}
int main()
{
    long long n;
    while(~scanf("%lld",&n))
    {
        if(n>1&&Miller_Rabin(n))
            puts("YES");
        else
            puts("NO");
    }
    return 0;
}
