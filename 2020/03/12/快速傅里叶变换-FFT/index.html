快速傅里叶变换-FFT
多项式 FFT
算法笔记
发布日期:   2020-03-12

前言

  快速傅里叶变换（Fast Fourier Transform，FFT）是一种可以在 $O(n\log n)$ 时间内完成的离散傅里叶变换（Discrete Fourier transform，DFT）算法，在算法竞赛中的主要应用之一是加速多项式乘法的计算。

  考虑到两个多项式 $ A(x),B(x) $ 的乘积 $ C(x) $ ，假设 $ A(x) $ 的项数为 $ n $ ，其系数构成的 $n$ 维向量为 $ (a_0,a_1,a_2,\cdots,a_{n-1}) $ ，$ B(x) $ 的项数为 $m$ ，其系数构成的 $ m $ 维向量为 $(b_0,b_1,b_2,\cdots,b_{m-1}) $。

  可以用这段代码表示：

for(int i=0;i<=n-1;i++)
    for(int j=0;j<=m-1;j++)
        c[i+j]=c[i+j]+a[i]*b[j];

  时间复杂度为 $ O(n^2) $。

多项式

系数表示法与点值表示法

  多项式的系数表示法：设 $A(x)$ 表示一个 $n-1$ 次多项式，所有项的系数组成的 $n$ 维向量 $(a_0,a_1,a_2,···,a_n)$ 唯一确定了这个多项式。
$$
A(x)=\sum\limits_{i=0}^{n-1}a_ix^i=a_0+a_1x+a_2x^2+···+a_{n-1}x^{n-1}
$$
  多项式的点值表示法：将一组互不相同的 $n$ 个 $x$ 带入多项式，得到的 $n$ 个值。设它们组成的 $n$ 维向量分别为 $(x_0,x_1,\cdots,x_{n-1}),(y_0,y_1,\cdots,y_{n-1})$。
$$
y_i=A(x_i)=\sum\limits_{j=0}^{n-1}a_jx_i^j
$$

求值与插值

  定理：一个 $n-1$ 次多项式在 $n$ 个不同点的取值唯一确定了该多项式。

  证明：假设命题不成立，存在两个不同的 $n-1$ 次多项式 $A(x)$、$B(x)$，满足对于任何 $i\in[0,n-1]$ ，有 $A(x_i)=B(x_i)$。

  令 $C(x)=A(x)-B(x)$，则 $C(x)$ 也是一个 $n-1$ 次多项式。对于任何 $i\in[0,n-1]$ ，有 $C(x_i)=0$。

  即 $C(x)$ 有 $n$ 个根，这与代数基本定理（一个 $n-1$ 次多项式在复数域上有且仅有 $n-1$ 个根）相矛盾，故 $C(x)$ 并不是一个 $n-1$ 次多项式，原命题成立，证毕。

  如果我们按照定义求一个多项式的点值表示，时间复杂度为 $O(n^2)$。

  已知多项式的点值表示，求其系数表示，可以使用插值。朴素的插值算法时间复杂度为 $O(n^2)$。

关于多项式的乘法

  定义两个多项式 $A(x)=\sum\limits_{i=0}^{n-1}a_ix^i$ 与 $B(x)=\sum\limits_{i=0}^{n-1}b_ix^i$ 相乘的结果为 $C(x)$ （假设两个多项式次数相同，若不同可在后面补零）。
$$
C(x)=A(x)\times B(x)=\sum\limits_{k=0}^{2n-2}(\sum\limits_{k=i+j}a_ib_j)x^k
$$
  两个 $n-1$ 次多项式相乘，得到的是一个 $2n-2$ 次多项式，时间复杂度为 $O(n^2)$。

  已知在一组插值节点 $(x_0,x_1,x_2,\cdots,x_{n-1})$ 中，$A(x),B(x)$ 的点值向量分别为 $(y_{A_0},y_{A_1},y_{A_2},\cdots,y_{A_{n-1}}),(y_{B_0},y_{B_1},y_{B_2},\cdots,y_{B_{n-1}})$，那么 $C(x)=A(x)·B(x)$ 的点值表达式为 $(y_{A_0}·y_{B_0},y_{A_1}·y_{B_1},y_{A_2}·y_{B_2},\cdots,y_{A_{n-1}}·y_{A_{B-1}})$，时间复杂度 $O(n)$ 。

  这也是快速傅里叶变换的基本思路，由 系数表达式 到 点值表达式 到 结果的点值表达式 再到 结果的系数表达式。

  把系数表达转换为点值表达的算法叫做 DFT。

  把点值表达转换为系数表达的算法叫做IDFT（DFT的逆运算）。

复数

定义

  我们把形如 $ a+bi $ 这样的数叫做复数，复数集合用 $C$ 来表示。其中 $ a $ 称为实部 $(\text{real part}) $ ，$ b $ 称为虚部 $ (\text{imaginary part}) $ ，$ i $ 为虚数单位，指满足 $ x^2=-1 $ 的一个解 $ \sqrt {-1} $ ；此外，对于这样对复数开偶次幂的数叫做虚数 $(\text{imaginary number})$。

复平面

  在复平面中，$ x $ 轴表示实数、$ y $ 轴（除原点外的所有点）代表虚数。每一个复数 $a+bi$ 对应复平面上一个从 $(0,0)$ 指向 $(a,b)$ 的向量。

  其中复数 $ z=a+bi $ 的模长 $ |z| $ 定义为 $z$ 在复平面的距离到原点的距离，$|z|=\sqrt{a^2+b^2}$ 。幅角 $ \theta $ 为实轴的正半轴正方向（逆时针）旋转到 $z$ 的有向角度。

  复数之间的运算满足结合律，交换律和分配律。

  复数相加遵循平行四边形定则， 复数相乘时，模长相乘，幅角相加。

单位根

  图中 $OA$ 为单位根，表示为 $ e^{i\theta} $，可知 $ e^{i\theta}=\cos \theta+i·\sin \theta $。

  下文中，如不特殊指明，均设 $n$ 为 $2$ 的正整数次幂。

  在复平面上，以原点为圆心，$1$ 为半径作圆，所得的圆叫做单位圆。以原点为起点，单位圆的 $n$ 等分点为终点，作 $n$ 个向量。设所得的幅角为正且最小的向量对应的复数为 $\omega_n$，称为 $n$ 次单位根。

  由复数乘法的定义（模长相乘，幅角相加）可知，其与 $n-1$ 个向量对应的复数分别为 $\omega_n^2,\omega_n^3,···,\omega_n^n$，其中 $w_n^n=w_n^0=1$。 它们可以由复数之间的乘法得来 $ \omega_n^{k}=\omega_{n}^{k-1}·\omega_{n}^{1}(2\leq k\leq n) $。

  单位根的幅角为周角的 $\frac{1}{n}$，这为我们提供了一个计算单位根及其幂的公式：
$$
\omega_n^k=\cos k\frac{2\pi}{n}+i·\sin k\frac{2\pi}{n}
$$

单位根的性质

  性质一(折半引理)：$\omega_{2n}^{2k}=\omega_n^k$

  从几何意义上看，在复平面上，二者表示的向量终点相同。

  更具体的，有：

$$
\cos\ 2k\frac{2\pi}{2n}+i·\sin\ 2k\frac{2\pi}{2n}=\cos\ k\frac{2\pi}{n}+i·\sin\ k\frac{2\pi}{n}
$$
  性质二(消去引理)：$\omega_n^{k+\frac{n}{2}}=-\omega_n^k$

  等式左边相当于 $ \omega_n^k $ 乘上 $ \omega_{n}^{\frac{n}{2}} $，考虑其值：

离散傅里叶变换（Discrete Fourier Transform）

  首先我们单独考虑一个 $n$ 项（$n=2^x$）的多项式 $A(x)$，其系数向量为 $(a_0,a_1,a_2,\cdots,a_{n-1})$。将 $ n $ 次单位根的 $ 0 $ ~ $ n-1 $ 次幂分别代入 $A(x)$ 得到其点值向量 $ (A(\omega_n^0),A(\omega_n^1),\cdots,A(\omega_n^{n-1})) $。这个过程称为离散傅里叶变换（Discrete Fourier Transform）。

  考虑将多项式按照系数下标的奇偶分为两部分：
$$
A(x)=(a_0+a_2x^2+a_4x^2+···+a_{n-2}x^{n-2})+(a_1x+a_3x^3+a_5x^5+···+a_{n-1}x^{n-1})
$$
  令
$$
A_1(x)=a_0+a_2x+a_4x^2+···+a_{n-2}x^{\frac{n}{2}-1}
$$

$$
A_2(x)=a_1+a_3x+a_5x^2+···+a_{n-1}x^{\frac{n}{2}-1}
$$

  则有：
$$
A(x)=A_1(x^2)+xA_2(x^2)
$$
  假设 $0\leq k\leq \frac{n}{2}-1,k\in Z $，现在要求 $A(\omega_n^k)$ ：
$$
A(\omega_n^k)=A_1(\omega_n^{2k})+\omega_n^kA_2(\omega_n^{2k})=A_1(\omega_{\frac{n}{2}}^k)+\omega_n^kA_2(\omega_{\frac{n}{2}}^k)
$$
  这一步转化利用了单位根的性质一。

  对于$\frac{n}{2}\leq k+\frac{n}{2}\leq n-1$：

  这一步转化除性质一外，还利用到了性质二和 $\omega_n^n=1$ 这一显然的结论。

  注意到，当 $k$ 取遍 $[0,\frac{n}{2}-1]$ 时，$k$ 和 $k+\frac{n}{2}$ 取遍了 $[0,n-1]$。

  也就是说，如果已知 $A_1(x)$ 和 $A_2(x)$ 在 $\omega_{\frac{n}{2}}^0,\omega_{\frac{n}{2}}^1,\cdots,\omega_n^{\frac{n}{2}-1}$ 处的点值，就可以在 $O(n)$ 的时间内求得 $A(x)$ 在 $\omega_n^0,\omega_n^1,···,\omega_n^{n-1}$ 处的取值。而关于 $A_1(x)$ 和 $A_2(x)$ 的问题都是相对于原问题规模缩小了一半的子问题，分治的边界为一个常数项 $a_0$。

  根据主定理，该分治算法的时间复杂度为：
$$
T(n)=2T(\frac{n}{2})+O(n)=O(n\log n)
$$
  这就是最常用的 FFT 算法——$\text{Cooley-Tukey}$ 算法。

离散傅里叶逆变换（Inverse Discrete Fourier Transform）

  使用快速傅里叶变换将点值表示的多项式转化为系数表示，这个过程叫做离散傅里叶反变换（Inverse Discrete Fourier Transform）。

  即由 $n$ 维点值向量 $(A(x_0),A(x_1),\cdots,A(x_{n-1}))$ 推出 $ n $ 维系数向量 $(a_0,a_1,\cdots,a_{n-1})$ 。

  设 $(d_0,d_1,\cdots,d_{n-1})$ 为 $(a_0,a_1,\cdots,a_{n-1}) $ 得到的离散傅里叶变换的结果。

  构造多项式 $F(x)=d_0+d_1·x+d_2·x^2+\cdots+d_{n-1}·x^{n-1}$。

  设向量 $(c_0,c_1,\cdots,c_{n-1})$ 中，$c_k$ 为 $F(x)$ 在 $x=\omega n^{-k}$ 的点值表示，即：
$$
c_k=\sum\limits{i=0}^{n-1}d_i(\omega_n^{-k})^i
$$
  将上式展开，得：

  考虑一个式子：
$$
S(\omega_n^k)=\sum\limits_{i=0}^{n-1}(\omega_{n}^{k})^i=1+\omega_n^k+(\omega_n^k)^2+(\omega_n^k)^3+\cdots+(\omega_n^k)^{n-1}
$$
  当 $k\neq 0$ 时，两边同时乘上 $ \omega_n^k $ 得：
$$
\omega_n^kS(\omega_n^k)=\omega_n^k+(\omega_n^k)^2+(\omega_n^k)^3+\cdots+(\omega_n^k)^{n}
$$
  两式相减，整理后得：
$$
\omega_n^kS(\omega_n^k)-S(\omega_n^k)=(\omega_n^k)^n-1
$$

$$
S(\omega_n^k)=\frac{(\omega_n^k)^n-1}{\omega_n^k-1}
$$

  分子为零，分母不为零，所以：$S(\omega_n^k)=0$

  当 $k=0$，显然 $ S(\omega_n^k)=n $。

  继续考虑上式：
$$
c_k=\sum\limits_{j=0}^{n-1}a_j\Big(\sum\limits_{i=0}^{n-1}(\omega_n^{j-k})^i\Big)=\sum\limits_{j=0}^{n-1}a_jS(\omega_n^{j-k})
$$
  当 $j=k$ 时，$S(\omega_n^{j-k})=n$，否则 $S(\omega_n^{j-k})=0$，即：
$$
c_k=na_k
$$

$$
a_k=\frac{1}{n}c_k
$$

  其中 $a_k$ 为原多项式 $A(x)$ 的系数向量 $(a_0,a_1,\cdots,a_k)$ 中的 $a_k$。

  所以，使用单位根的倒数代替单位根，做一次类似快速傅里叶变换的过程，再将结果每个数除以 $n$，即为傅里叶逆变换的结果。

  对于多项式 $A(x)$ 由插值节点 $ (\omega_{n}^{0},\omega_{n}^{1},\omega_{n}^{2},\cdots,\omega_{n}^{n-1}) $ 做离散傅里叶变换得到的点值向量 $ (d_0,d_1,\cdots,d_{n-1}) $。我们将 $(\omega_{n}^{0},\omega_{n}^{-1},\omega_{n}^{-2},\cdots,\omega_{n}^{-(n-1)}) $ 作为插值节点，$ (d_0,d_1,\cdots,d_{n-1}) $ 作为系数向量，做一次离散傅里叶变换得到的向量每一项都除以 $ n $ 之后得到的 $ \big(\frac{c_0}{n},\frac{c_1}{n},\cdots,\frac{c_{n-1}}{n} \big) $ 就是多项式的系数向量 $ (a_0,a_1,\cdots,a_{n-1})$。

FFT在C++中的实现

  C++ 的 STL 在头文件 $ \text{complex} $ 中提供一个复数的模板实现 $\text{std::complex}$，其中 $T$ 为实数类型，一般取 $\text{double}$ ，在对精度要求较高的时候可以使用 $ \text{long double} $ 或 $ \text{__float128} $（不常用）。

  考虑到单位根的倒数等于其共轭复数，在程序实现中，为了减小误差，通常使用 $ \text{std::conj()} $ 取得 IDFT 所需的单位根的倒数。

复数类

struct Complex  
{
    double r,i;
    Complex(){}
    Complex(double r,double i):r(r),i(i){}
    inline void real(const double &x){r=x;}
    inline double real(){return r;}
    inline Complex operator + (const Complex &rhs) const{
        return Complex(r+rhs.r,i+rhs.i);
    }
    inline Complex operator - (const Complex &rhs) const{
        return Complex(r-rhs.r,i-rhs.i) ;
    }
    inline Complex operator * (const Complex &rhs) const{
        return Complex(r*rhs.r-i*rhs.i,r*rhs.i+i*rhs.r) ;
    }
    inline void operator /= (const double &x){
        r/=x,i/=x;
    }
    inline void operator *= (const Complex &rhs){
        *this=Complex(r*rhs.r-i*rhs.i,r*rhs.i+i*rhs.r) ;
    }
    inline void operator += (const Complex &rhs){
        r+=rhs.r,i+=rhs.i;
    }
    inline Complex conj(){
        return Complex(r,-i);
    }
};

递归实现

  由上面的分析可以得到一个递归的写法。

bool inverse=false;
inline Complex omega(const int &n,const int &k)  
{
    if(!inverse) return Complex(cos(2*PI/n*k),sin(2*PI/n*k));
    return Complex(cos(2*PI/n*k),sin(2*PI/n*k)).conj();
}
inline void FFT(Complex *a,const int &n)  
{
    if(n==1) return ;
    static Complex buf[MAXN];
    const int m=n>>1;
    //按照系数奇偶划分为两半 
    for(int i=0;i<m;++ i )  
    {
        buf[i]=a[i<<1];
        buf[i+m]=a[i<<1|1];
    }
    memcpy(a,buf,sizeof(Complex)*(n+1));
    //分治 
    Complex *a1=a,*a2=a+m;
    FFT(a1,m);
    FFT(a2,m);
    //合并两个子问题 
    for(int i=0;i<m;i++)  
    {
        Complex x=omega(n,i);
        buf[i]=a1[i]+x*a2[i];
        buf[i+m]=a1[i]-x*a2[i];
    }
    memcpy(a,buf,sizeof(Complex)*(n+1));
}

迭代实现

  递归实现的 FFT 需要用到辅助数组，并且常数比较大。

  可以发现，分治到边界后的下标等于原下标的二进制位翻转。

  代码实现，枚举每个二进制位即可。

int k=0;
while((1<<k)<n) 
    k++;
for(int i=0;i<n;i++) 
{
    int t=0;
    for(int j=0;j<k;j++) 
        if(i&(i<<j)) 
            t=t|(1<<(k-j-1));
    if(i<t) 
        std::swap(a[i],a[t]);
}

蝴蝶操作

  考虑合并两个子问题的过程，假设 $A_1(\omega_{\frac{n}{2}}^{k})$ 和 $A_2(\omega_{\frac{n}{2}}^{k})$ 分别存在 $\text{a}[k]$ 和 $\text{a}[\frac{n}{2}+k]$ 中，$A(\omega_{n}^{k})$ 和 $A(\omega_{n}^{k})$ 和 $A(\omega_{n}^{k+\frac{n}{2}})$ 将要被存放在 $ \text{buf}[k] $ 和 $ \text{buf}[\frac{n}{2}+k] $ 中，合并的单位操作可表示为：
$$
\text{buf}[k]\leftarrow a[k]+\omega_{n}^{k}\times a[\frac{n}{2}+k]
$$

$$
\text{buf}[\frac{n}{2}+k]\leftarrow a[k]-\omega_{n}^{k}\times a[\frac{n}{2}+k]
$$

  考虑加入一个临时变量 $ t=\omega_{n}^{k}\times a[\frac{n}{2}+k] $ ，使得这个过程可以在原地完成，不需要另一个数组 $b$ ，也就是说，将 $A(\omega_{n}^{k}) $ 和 $A(\omega_{n}^{k+\frac{n}{2}})$ 存放在 $a[k]$ 和 $a[\frac{n}{2}+k]$ 中，覆盖原来的值。

 &esmp;这一过程被称为蝴蝶操作。

代码

  $\text{omega[k]}$ 中保存 $\omega_{n}^{k}$（IDFT时保存 $\omega_{n}^{-k}$）。

  枚举 $l$，表示一次要将 $\frac{l}{2}$ 长度的序列答案合并为长度为 $l$ 的，根据单位根的性质一，有 $ \omega_{l}^{k}=\omega_{n}^{\frac{n}{l}k} $。

for(int l=2;l<=n;l<<=1)
    {
        int m=l/2;
        for(Complex *p=a;p!=a+n;p+=l)
        {
            for(int i=0;i<m;i++)
            {
                Complex t=omega[n/l*i]*p[m+i];
                p[m+i]=p[i]-t;
                p[i]+=t;
            }
        }
    }

模板

struct FastFourierTransform
{
    Complex omega[N],omegaInverse[N];
    void init(const int &n)
    {
        for(int i=0;i<n;i++)
        {
            omega[i]=Complex(cos(2*PI/n*i),sin(2*PI/n*i));
            omegaInverse[i]=omega[i].conj();
        }
    }
    void transform(Complex *a,const int &n,const Complex *omega)
    {
        for(int i=0,j=0;i<n;i++)  
        {
            if(i>j)  
                std::swap(a[i],a[j]) ;
            for(int l=n>>1;(j^=l)<l;l>>=1);
        }
        for(int l=2;l<=n;l<<=1)
        {
            int m=l/2;
            for(Complex *p=a;p!=a+n;p+=l)
            {
                for(int i=0;i<m;i++)
                {
                    Complex t=omega[n/l*i]*p[m+i];
                    p[m+i]=p[i]-t;
                    p[i]+=t;
                }
            }
        }
    }
    void dft(Complex *a,const int &n)
    {
        transform(a,n,omega);
    }
    void idft(Complex *a,const int &n)
    {
        transform(a,n,omegaInverse) ;
        for(int i=0;i<n;i++) 
            a[i]/=n;
    }
}FFT;

例题

P3803 【模板】多项式乘法（FFT）

题目描述

给定一个 $n$ 次多项式 $F(x)$，和一个 $m$ 次多项式 $G(x)(1\leq n,m,\leq 10^6)$。

请求出 $F(x)$ 和 $G(x)$ 的卷积。

输入格式

第一行两个整数 $n,m$。

接下来一行 $n+1$ 个数字，从低到高表示 $F(x)$ 的系数。

接下来一行 $m+1$ 个数字，从低到高表示 $G(x)$ 的系数。

输出格式

一行 $n+m+1$ 个数字，从低到高表示 $F(x)·G(x)$ 的系数。

代码

#include<bits/stdc++.h>
using namespace std;
const double PI=acos(-1.0); 
const int N=1000010;
inline int read()
{
    int f=1,ans=0;
    char ch;
    ch=getchar();
    while(ch<'0'||ch>'9') 
    {
        if(ch=='-')
            f=-1;
        ch=getchar(); 
    } 
    while(ch>='0'&&ch<='9') 
    {
        ans=ans*10+ch-'0'; 
        ch=getchar();
    }
    return ans*f; 
}

        return Complex(r+rhs.r,i+rhs.i);
    }
    inline Complex operator - (const Complex &rhs) const{
        return Complex(r-rhs.r,i-rhs.i) ;
    }
    inline Complex operator * (const Complex &rhs) const{
        return Complex(r*rhs.r-i*rhs.i,r*rhs.i+i*rhs.r) ;
    }
    inline void operator /= (const double &x){
        r/=x,i/=x;
    }
    inline void operator *= (const Complex &rhs){
        *this=Complex(r*rhs.r-i*rhs.i,r*rhs.i+i*rhs.r) ;
    }

    }

}A[N<<2],B[N<<2],C[N<<2];
int rev[N<<2];
int n,m,limit,k;
void FFT(Complex *a,int n,int DFT)
{
    for(int i=0;i<n;i++)
        if(i<rev[i])
            swap(a[i],a[rev[i]]);    
    for(int s=1;(1<<s)<=n;s++)
    {
        int m=(1<<s);
        Complex wn=Complex(cos(2*PI/m),DFT*sin(2*PI/m));
        for(int t=0;t<n;t=t+m)
        {
            Complex w=Complex(1,0);
            for(int j=0;j<(m>>1);j++)
            {
                Complex u=a[t+j],v=w*a[t+j+(m>>1)];
                a[t+j]=u+v;
                a[t+j+(m>>1)]=u-v;
                w=w*wn;
            }
        }
    }
    if(DFT==-1)
    {
        for(int i=0;i<n;i++)
            a[i]/=n;
    }
}
int main()
{
    n=read(),m=read();
    for(int i=0;i<=n;i++)
        A[i].r=read();
    for(int i=0;i<=m;i++)
        B[i].r=read();
    m=m+n;
    limit=0;
    for(limit=1;limit<=m;limit<<=1)
        k++;
    for(int i=0;i<limit;i++)
        rev[i]=(rev[i>>1]>>1)|((i&1)<<(k-1));
    FFT(A,limit,1);
    FFT(B,limit,1);
    for(int i=0;i<=limit;i++)
        C[i]=A[i]*B[i];
    FFT(C,limit,-1);
    for(int i=0;i<=m;i++)
        printf("%d ",(int)(C[i].r+0.5));
    cout<<endl;
    return 0;
}

struct FastFourierTransform
{
    Complex omega[N],omegaInverse[N];
    void init(const int &n)
    {
        for(int i=0;i<n;i++)
        {
            omega[i]=Complex(cos(2*PI/n*i),sin(2*PI/n*i));
            omegaInverse[i]=omega[i].conj();
        }
    }
    void transform(Complex *a,const int &n,const Complex *omega)
    {
        for(int i=0,j=0;i<n;i++)  
        {
            if(i>j)  
                std::swap(a[i],a[j]) ;
            for(int l=n>>1;(j^=l)<l;l>>=1);
        }
        for(int l=2;l<=n;l<<=1)
        {
            int m=l/2;
            for(Complex *p=a;p!=a+n;p+=l)
            {
                for(int i=0;i<m;i++)
                {
                    Complex t=omega[n/l*i]*p[m+i];
                    p[m+i]=p[i]-t;
                    p[i]+=t;
                }
            }
        }
    }
    void dft(Complex *a,const int &n)
    {
        transform(a,n,omega);
    }
    void idft(Complex *a,const int &n)
    {
        transform(a,n,omegaInverse) ;
        for(int i=0;i<n;i++) 
            a[i]/=n;
    }
}FFT;

    return 0;
}
