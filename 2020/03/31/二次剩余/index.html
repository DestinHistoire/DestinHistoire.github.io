二次剩余
数论 二次剩余
算法笔记
发布日期:   2020-03-31

  若存在 $x\in[1,p-1]$，使方程 $x^2\equiv a\pmod p$ 有解，则称 $a$ 为 $p$ 的二次剩余，否则 $a$ 为 $p$ 的非二次剩余（$p$ 为奇素数）。

勒让德符号

  定义勒让德符号为：

欧拉判别准则

  对于勒让德符号，有：
$$
\bigg( \frac{a}{p}\bigg)\equiv a^{\frac{p-1}{2}}\pmod p
$$
  由费马小定理：$ a^{p-1}\equiv 1\pmod p$，且 $p-1$ 是偶数：
$$
(a^{\frac{p-1}{2}}-1)(a^{\frac{p-1}{2}}+1)\equiv 0\pmod p
$$
  $p$ 是一个质数，所以 $a^{\frac{p-1}{2}}-1$ 和 $a^{\frac{p-1}{2}}+1$ 中必有一个是 $p$ 的倍数，因此 $a^{\frac{p-1}{2}}\equiv 1\pmod p$ 或 $a^{\frac{p-1}{2}}\equiv -1\pmod p$。

二次剩余的充要条件：勒让德符号为 $1$

充分性

  若 $a$ 在模 $p$ 意义下是二次剩余，则 $a^{\frac{p-1}{2}}\equiv 1\pmod p$。

证明：

  若 $a$ 在模 $p$ 意义下是二次剩余，即方程 $x^2\equiv a\pmod p$ 有解，而 $p$ 与 $a,x$ 都互质。

  根据费马小定理：
$$
a^{\frac{p-1}{2}}\equiv x^{p-1}\equiv 1\pmod p
$$

必要性

  若 $a^{\frac{p-1}{2}}\equiv 1\pmod p$，则 $a$ 是模 $p$ 的二次剩余。

证明：

  $p$ 是一个奇素数，所以关于 $p$ 的原根存在。设 $g$ 是 $p$ 的一个原根，则存在 $j\in[1,p-1]$，使得 $a=g^j$。

  所以：$g^{j\frac{p-1}{2}}\equiv 1\pmod p$。

  $g$ 是 $p$ 的一个原根，因此 $g$ 模 $p$ 的指数为 $p-1$，所以 $(p-1)\Big| \frac{j(p-1)}{2}$。说明 $j$ 是一个偶数，令 $i=\frac{j}{2}$，有 $(g^i)^2=g^{2i}=a$，因此 $a$ 是模 $p$ 意义下的二次剩余。

二次非剩余的充要条件：勒让德符号为 $-1$

证明：

  由于 $p$ 是质数，由 扩展欧几里得算法 可知对于每个 $1\leq i\leq p-1$，都存在唯一的一个整数 $j$，使得 $ij\equiv a\pmod p$。所以可以把 $1,2,\cdots ,p-1$ 分成 $\frac{p-1}{2}$ 对，使得每对的乘积都为 $a$。此时有 $ (p-1)!\equiv a^{\frac{p-1}{2}}$，根据威尔逊定理有 $ (p-1)!\equiv-1\pmod p$，所以 $a^{\frac{p-1}{2}}\equiv -1\pmod p$。

  涉及 扩展欧几里得算法 的证明：

  由于 $ij\equiv a\pmod p$，则 $ij+pk=a$。因为 $\gcd(i,p)=1$，所以 $ij+pk=1$ 必然有解。因为 $a$ 是 $1$ 的倍数，所以 $ij+pk=n$ 必然有解。

  令 $j=j_0+pk’$，则有且只有一个解 $j$ 落在 $[1,p-1]$ 内。

$p|a$ 时，勒让德符号为 $0$

  显然成立。

相关定理

定理 $1$

  对于方程 $x^2\equiv a\pmod p$，共有 $\frac{p-1}{2}$ 个不同的 $a$ 使该方程有解。

证明：考虑除 $0$ 外所有的 $x^2$，如果存在两个不同的数 $u,v$ 满足 $u^2\equiv v^2\pmod p$，显然有 $p\mid(u^2-v^2)$，由平方差公式：$p\mid(u-v)(u+v)$，由于 $u,v\leq p-1$，则 $u-v\in[1,p-2]$，显然 $p\nmid(u-v)$，所以 $p\mid (u+v)$，即 $u+v\equiv 0\pmod p$。说明对于每对 $u+v=p$，$u^2\equiv a\pmod p$ 且 $v^2\equiv a\pmod p$。因此共有 $\frac{p-1}{2}$ 个不同的 $a$ 使该方程有解，每个 $x^2\equiv a\pmod p $ 恰有两个解，且两个解的和为 $p$。

定理 $2$

$$
(a+b)^p\equiv a^p+b^p\pmod p
$$

证明：

  左边按照二项式定理展开：$(a+b)^p=\sum\limits_{i=0}^{p}C_{p}^{i}a^{i}b^{p-i}$。

  当 $i\neq 0$ 且 $i\neq p$ 时，$C_{p}^{i} \mod p=0$，所以只剩下第一项和最后一项，即 $a^p$ 和 $ b^p $。

Cipolla算法

  现在要求出方程 $x^2\equiv n\pmod p $ 的一个解。

  在 $[0,p-1]$ 中随机挑选一个数 $a$，定义 $\omega=\sqrt{a^2-n}$，若 $(a^2-n)^{\frac{p-1}{2}}\equiv -1\pmod p$，即 $\Big(\frac{\omega^2}{p}\Big)=-1$，那么 $x=(a+\omega)^{\frac{p+1}{2}}$。

证明：

  可以发现：
$$
(a^2-n)^{\frac{p-1}{2}}\equiv -1\pmod p\Longrightarrow (\omega ^2)^{\frac{p-1}{2}}\equiv -1\pmod p\Longrightarrow \omega^{p-1}\equiv-1\pmod p
$$
  则：

  证毕。

  由于满足 $\Big(\frac{\omega^2}{p}\Big)=-1$ 的解有 $\frac{p-1}{2}$ 个，选中的概率为 $\frac{1}{2}$，所以期望是两次。

  $x$ 是一个解，$p-x$ 显然也是解，因为 $(p-x)^2=p^2-2\ast p\ast x+x^2\equiv x^2\pmod p$。所以求出了一个解，相当于求出了另一个。

扩域

  上文中，$a^2-n$ 不能直接开根号，但是可以把域扩充一下，设扩充后的域为 $\mathbb{F}{p^2}$，定义 $\omega=\sqrt{a^2-n}$ 为这个域的 “虚数单位元”（类比 $i=\sqrt{-1}$），则这个 “复数域” $\mathbb{F}{p^2}$ 中的数可以表示为 $a+b\omega$（类比复数域 $a+bi$，$b\omega$ 相当于虚部）。

  我们将复数域的四则运算法则全部类比到 $\mathbb{F}_{p^2}$ 上，显然它依然满足封闭性、交换律、结合律、分配律，还存在加法零元和乘法逆元。具体见下图：

  由此定义后，$a^2-n$ 就可以开根号了。

  最后一个问题：我们需要的解是在 $\mathbb{F}p$ 意义下的，但是求出的解是在 $\mathbb{F}{p^2}$ 意义下的，也就是说：如何确定求出的解虚部为 $0$ ？

  根据 拉格朗日定理，可知在在任意一个模 $p(p\in\mathbb{P})$ 的数域里面，任意一个多项式 $f(x)$ 最多有$\text{deg}(f(x))$ 个根（ $f(x)\equiv 0\pmod p$）的解称为 $f(x)$ 在 $\mathbb{F}p $ 下的根），$\text{deg}$ 表示多项式的度数，即最大指数。由于 $\mathbb{F}{p^2}$ 是对 $ \mathbb{F}p $ 域的扩充，$\mathbb{F}_p$ 域的两根一定在 $\mathbb{F}{p^2} $ 内也有效，并且我们知道 $x^2-n$ 在数域 $\mathbb{F}_{p^2}$ 下的根有两个 $(x_1,x_2)$，那么 $x_1,x_2$ 一定也是 $\mathbb{F}_p$ 域下的根，也就是 “虚部” 系数为 $0$。

  算法时间复杂度为 $O(\log p)$。

例题

P5491 【模板】二次剩余

题目描述

  求解方程 $x^2 \equiv n\pmod p$。多组数据。保证 $p$ 是奇素数。

输入格式

  第一行一个整数 $T$ 表示数据组数。

  接下来 $T$ 行，每行一个 $n$，一个 $p$。

  $T\leq 10000,n,p\leq 10^9+9$。

输出格式

  输出共 $T$ 行。

  对于每一行输出：

  若有解，则按  $\bmod  p$ 后递增的顺序输出在 $ \bmod  p$ 意义下的全部解。

  若两解相同，只输出其中一个。

  若无解，则输出 $\text{Hola!}$。

代码

#include<bits/stdc++.h>
using namespace std;
long long n,p,w_2;
struct Complex
{
    long long x,y;
    Complex(long long _x,long long _y){
        x=_x,y=_y;
    }
};
Complex operator * (Complex a,Complex b)
{
    return Complex((a.x*b.x%p+w_2*a.y%p*b.y%p+p)%p,(a.x*b.y%p+a.y*b.x%p+p)%p);
}
bool operator == (Complex a,Complex b)
{
    return a.x==b.x&&a.y==b.y;
}
long long quick_pow(long long a,long long b)
{
    long long ans=1;
    while(b)
    {
        if(b&1)
            ans=ans*a%p;
        a=a*a%p;
        b>>=1;
    }
    return ans;
}
Complex quick_pow(Complex a,long long b)
{
    Complex ans=Complex(1,0);
    while(b)
    {
        if(b&1)
            ans=ans*a;
        a=a*a;
        b>>=1;
    }
    return ans;
}
long long Cipolla()
{
    long long a;
    while(true)
    {
        a=rand()%p;
        long long w=(a*a%p-n+p)%p;
        if(quick_pow(w,(p-1)/2)==p-1)
            break;
    }
    w_2=(a*a%p-n+p)%p;
    return a;
} 
int main()
{
    srand(time(0));
    int T;
    cin>>T;
    while(T--)
    {
        cin>>n>>p;
        n=n%p;
        if(n==0)
            cout<<0<<endl;
        else if(quick_pow(n,(p-1)/2)==p-1)
            puts("Hola!");
        else
        {
            long long ans=quick_pow(Complex(Cipolla(),1),(p+1)/2).x;
            if(ans>p-ans)
                ans=p-ans;
            cout<<ans<<" "<<p-ans<<endl;            
        }
    }
    return 0;
}
